# 大气折射类型

## 定义

在球面分层大气中，根据地-空链路射线与地球表面相对位置关系划分的折射类别。分类依据为[[射线曲率半径]] $\rho_{curv}$ 与地球半径 $r_{earth}$ 的比值。

## 七种类型

| 类型 | $\rho_{curv}/r_{earth}$ | $dN/dz$ $(m^{-1})$ | $dM/dz$ $(m^{-1})$ | 射线行为 |
|------|-------------------------|---------------------|--------------------|----------|
| [[负折射]] | < 0 | > 0 | > 0.157 | 射线向上凹 |
| [[无折射]] | $\pm \infty$ | 0 | 0.157 | 直线传播 |
| 次折射 | > 4, < $\infty$ | < 0 | < 0.157 | 轻微向地面弯曲 |
| [[标准折射]] | 4 | -0.077~0 | 0.080~0.157 | 向地面弯曲，可穿出对流层 |
| [[过折射]] | 1~4 | -0.157~-0.077 | 0~0.080 | 向地面弯曲但不回到地面 |
| [[临界折射]] | 1 | -0.157 | 0 | 射线平行地面滑行 |
| [[陷获折射]] | < 1 | < -0.157 | < 0 | 射线返回地面形成波导 |

## 核心判据

$\rho_{curv}/r_{earth}$ 由[[折射指数 N|折射指数]]梯度决定。折射率梯度越负 → 射线越向地面弯曲。
