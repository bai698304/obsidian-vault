# 射线曲率半径

## 定义

描述电磁波射线在弯曲传播路径上各点弯曲程度的参数，符号 $\rho_{curv}$。该点处的曲线与半径为 $\rho_{curv}$ 的某段圆弧重合。

## 精确表达式

$$\rho_{curv} = - \left\{ \frac{n(h_{antenna})(r_{earth} + h_{antenna}) \cos \theta_{elevation\_0}}{[n(h)]^2 (r_{earth} + h)} \frac{dn(h)}{dh} \right\}^{-1}$$

## 简化近似

对流层高度相对地球半径可忽略时：

$$\rho_{curv} = - \left[ \frac{\cos \theta_{elevation\_0} \, dn(h)}{dh} \right]^{-1}$$

此时曲率半径完全由**折射率梯度**和**初始仰角**决定。

## 折射类型判据

$\rho_{curv}$ 与地球半径 $r_{earth}$ 的比值决定了[[大气折射类型]]：

| $\rho_{curv}/r_{earth}$ | 折射类型 |
|--------------------------|----------|
| < 0 | [[负折射]] |
| $\pm \infty$ | [[无折射]] |
| = 4 | [[标准折射]] |
| 1~4 | [[过折射]] |
| = 1 | [[临界折射]] |
| < 1 | [[陷获折射]] |
