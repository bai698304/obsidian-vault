# 交叉极化隔离度 XPI

## 定义

用同一频率同时传输两路互为正交的极化信号时，一种极化波的**同极化分量**与另一种极化波的**交叉极化分量**在接收点的功率比。值越大，去极化效应越小。

## 左旋圆极化 XPI

$$\mathrm{XPI_L} = 20\lg \frac{E_{LL}}{E_{LR}}$$

- $E_{LL}$：左旋发射 → 左旋接收（同极化）
- $E_{LR}$：右旋发射 → 左旋接收（交叉极化）

## 与 XPD 的细微差异

| 参数 | 测量条件 |
|------|----------|
| [[交叉极化分辨率 XPD\|XPD]] | 发射**单一极化**信号 |
| **XPI** | 同时发射**两路正交极化**信号，考虑通道间干扰 |

## XPD 或 XPI 越大 → 去极化效应越小
