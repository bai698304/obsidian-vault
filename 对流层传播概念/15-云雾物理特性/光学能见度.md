# 光学能见度

$V_b$：表征云、雾强度特征的参数，与可见光的[[衰减]]系数成反比。

$$V_b = \frac{1}{\alpha_{atten}} \ln\left|\frac{1}{K_{threshold}}\right|$$

其中 $K_{threshold}$ 为理想黑体相对于理想白背景的门限对比度（通常 0.018~0.03），$\alpha_{atten}$ 为 0.55 μm 波长衰减系数。

**能见距离 $V_{bm}$**：$K_{threshold} = 0.02$ 时的能见度。

$V_b$ 越小或[[含水量]] $C_w$ 越高 → 云雾越强。

**雾按 $V_b$ 分级**：重雾(≤50m) / 浓雾(50~200) / 大雾(200~500) / 雾(500~1000) / 轻雾(>1000m)
