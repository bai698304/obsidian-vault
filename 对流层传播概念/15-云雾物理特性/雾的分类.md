# 雾的分类

## 按能见度
重雾(≤50m) / 浓雾(50~200m) / 大雾(200~500m) / 雾(500~1000m) / 轻雾或霭(>1000m)。

## 按形成机制
| 类型      | 成因                                    |
| ------- | ------------------------------------- |
| **辐射雾** | 空气因==辐射冷却达到过饱和==，多出现于晴朗微风夜间/早晨。秋冬季节常见 |
| **平流雾** | 暖湿空气平流到冷下垫面上冷却。春夏沿海海雾典型               |
| **蒸气雾** | 冷空气流经暖水面，蒸发使冷空气水汽增加过饱和                |
| **上坡雾** | 潮湿空气沿山坡上升，绝热膨胀冷却。青藏高原/云贵高原常见          |
| **锋面雾** | 暖气团降水落入冷空气层，水滴蒸发使冷空气过饱和。梅雨季节常见        |
| **湿雾**  | 有雾同时伴有雾滴向地面有序下降                       |

## 按含水量
干雪 / 略潮雪 / 潮雪 / 湿雪 / 水雪（实际降雪分类）
