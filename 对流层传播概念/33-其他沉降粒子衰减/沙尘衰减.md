# 沙尘衰减

[[沙尘暴|风沙天气]]中悬浮沙尘粒子对电磁波的吸收和散射导致衰减。6.1.3 节通用理论适用。

**两种等效方法**：
- 尘土粒子 → [[Mie理论|球形等效]]
- 沙粒 → **椭球形等效**（轴比 $a:b:c = 1:0.75:0.75^2$）

**球形等效 Mie 衰减率**：
- 吸收项 $\propto a^3/\lambda$（主导）
- 散射项 $\propto a^6/\lambda^4$（可忽略）

简化公式：
$$\gamma_{dB/km} = -\frac{0.6287 f}{V_b} a_e \mathrm{Im}\left[\frac{(\varepsilon_r - j\varepsilon_{r\_Im}) - 1}{(\varepsilon_r - j\varepsilon_{r\_Im}) + 2}\right]$$

- $a_e$：[[加权等效半径]]
- $V_b$：[[光学能见度|能见度]]

**椭球形等效**需用 Rayleigh 近似计算水平/垂直极化的[[散射振幅矢量与散射函数矢量|前向散射振幅]]。
