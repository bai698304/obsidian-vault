# 椭球极化因子

描述椭球形粒子三个主轴方向极化特性的参数 $A_i$（$i=1,2,3$）：

$$A_i = \frac{abc}{2} \int_0^\infty \frac{ds}{(s + a_i^2) \sqrt{(s+a)^2 (s+b)^2 (s+c)^2}}$$

满足 $A_1 + A_2 + A_3 = 1$。

对于**小椭球**近似：
$$A_1 : A_2 : A_3 = a^{-1} : b^{-1} : c^{-1}$$

在[[沙尘衰减]]和[[固态水凝物衰减]]的 Rayleigh 近似中，通过 $A_i$ 计算前向散射振幅 $f_i(0)$，进而得到[[散射振幅矢量与散射函数矢量|散射函数]]。
