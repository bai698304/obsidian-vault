# ITU-R 大气吸收模型

## 定义

ITU-R P.676 系列建议书给出的计算[[大气吸收衰减率]]的模型，分为**经验模型**和**精确模型**两类。

## 经验模型 (ITU-R P.676-9)

$$\gamma_a = \gamma_O + \gamma_V$$

其中 $\gamma_O$ 和 $\gamma_V$ 分别为[[氧气吸收衰减率]]和[[水汽吸收衰减率]]，涉及参数：
- $r_p = p / 1013$（归一化大气压）
- $r_t = 288 / (t + 273)$（归一化温度）
- $\rho_V$：水蒸气含量密度 (g/m³)

经验模型适用频率：**1~350 GHz**。

## 地-空链路简化模型

$$A_{atmosphere} = A_O + A_V = \gamma_O h_{O\_scale} + \gamma_V h_{V\_scale}$$

其中 $h_{O\_scale}$ 和 $h_{V\_scale}$ 分别为氧气和水汽的**标高**（等效均匀大气高度），近似认为从地面至该高度范围内大气均匀分布。

积分高度上限 $h_{critical}$：至少 **30 km**，一般要求 **100 km**。
