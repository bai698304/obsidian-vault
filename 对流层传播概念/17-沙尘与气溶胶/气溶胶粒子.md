# 气溶胶粒子

对流层中除雨/雾/云/冰晶/冰雹/雪/沙尘以外的自然和人工生成的固体与液体颗粒的复杂动力混合物。尺寸约 **10⁻³ ~ 10² μm**。

**按来源**：
- 天然气溶胶：光化学反应/生物活动/海浪/火山爆发/宇宙空间运动
- 人为气溶胶：切割/燃煤/电弧焊等生活生产活动

**按粒径分类**：
| 类别 | 粒径 |
|------|------|
| 亚显气溶胶 | < 0.25 μm |
| 显微气溶胶 | 0.25~10 μm |
| 细型气溶胶 | 10~40 μm |
| 粗气溶胶 | > 40 μm |

**按尺度分类**：爱根核(<0.1 μm) / 大粒子(0.1~1 μm) / 巨粒子(>1 μm)

**按滞留时间**：降沉气溶胶(>10 μm) / 悬浮气溶胶(<10 μm)
