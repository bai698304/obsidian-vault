# 沙尘粒径分布

描述沙尘环境中不同半径粒子群的分布情况。有 **6 种**常用概率密度函数：

| 分布类型 | 关键参数 |
|---------|---------|
| **均匀分布** | $a_{dust\_min}, a_{dust\_max}$ |
| **指数分布** | 平均半径 $\lambda_{dust\_e}$ |
| **麦克斯韦分布** | 平均半径 $\lambda_{dust\_m}$ |
| **瑞利分布** | $\lambda_{dust\_r}$（半径平方均值一半） |
| **正态分布** | 均值 $r_{dust\_0}$ + 方差 $\lambda_{dust\_r}^2$ |
| **对数正态分布** | $\langle \ln r_{dust} \rangle$ 均值和方差 |

我国腾格里沙漠和黄河沙滩有实际测量参数。不同地区/不同次沙尘事件/同一事件不同阶段分布差异较大。
