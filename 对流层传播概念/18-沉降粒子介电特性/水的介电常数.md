# 水的介电常数

水的复介电常数 $\varepsilon_r = \varepsilon_{Re} - j\varepsilon_{Im}$ 是频率和温度的复杂函数。

- 波长 **> 1 mm**：分子极化理论（Debye 模型）
- 波长 **< 1 mm**：量子谐振理论

**主要模型**：

1. **Debye 模型**：$\varepsilon_r = \varepsilon_\infty + \frac{\varepsilon_s - \varepsilon_\infty}{1 + j\lambda_s/\lambda}$
   - 参数 $\varepsilon_s, \varepsilon_\infty, \lambda_s$ 均与温度密切相关

2. **Ray 经验公式**（ITU-R 雨衰模型采用）：实部和虚部分别表示，适用于 -20°C~50°C

3. **双 Debye 模型**（云雾传播特性计算常用）：含两级弛豫频率 $f_p, f_s$，适用 0~1000 GHz

4. **$n$-Debye 模型**：多级弛豫的广义模型（尚在探索中）

**适用性边界**：
- $f \leq 100$ GHz：标准 Debye/Ray 公式可靠
- $100 < f \leq 1000$ GHz：双 Debye 或修正参数 Debye
- $1 < f \leq 30$ THz：需加入洛伦兹共振项（分子伸缩振动 + 分子间振动）
