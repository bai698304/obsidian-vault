# 洛伦兹共振

当电磁波频率远大于分子弛豫频率时，分子"电"结构除"极化"运动外，还出现振动、伸缩、旋转等运动形态，需用量子力学理论分析。

在 1~30 THz 频段，水的介电常数需加入洛伦兹共振贡献：
$$\varepsilon(f) = \varepsilon_M(f) + \varepsilon_R(f)$$

其中：
- $\varepsilon_M(f)$：分子极化理论结果（双 Debye 模型）
- $\varepsilon_R(f)$：两个洛伦兹共振过程的贡献

$$\varepsilon_R(f) = \sum_{i=1}^{2} \left[\frac{A_{R\_i}}{f_{R\_i}^2 - f^2 + i(\Delta f_{R\_i})f} - \frac{A_{R\_i}}{f_{R\_i}^2}\right]$$

两个中心频率：$f_{R\_1} = 5.11$ THz（分子伸缩振动）、$f_{R\_2} = 18.2$ THz（分子间振动）。
