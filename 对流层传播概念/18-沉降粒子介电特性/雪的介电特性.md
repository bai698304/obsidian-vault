# 雪的介电特性

雪是冰晶、水和空气的混合物，采用**等效均匀媒质介电特性模型**描述。

**Rayleigh 等效模型**：
$$\frac{\varepsilon_{eff} - 1}{\varepsilon_{eff} + u} = p_1\frac{\varepsilon_1-1}{\varepsilon_1+u} + p_2\frac{\varepsilon_2-1}{\varepsilon_2+u} + p_3\frac{\varepsilon_3-1}{\varepsilon_3+u}$$
形状数 $u$：干雪 = 2，潮雪 = 9，湿雪 = 20。

**干雪**（冰晶+空气）：
- $\varepsilon_{Re} = (1 + 0.508\rho_s)^3$
- $\varepsilon_{Im}$ 仅与冰的虚部有关

**湿雪**（3~37 GHz，湿度 1%~12%）：
- $\varepsilon_{Re\_w} = A + \frac{B m_V^x}{1 + (f/f_0)^2}$
- $\varepsilon_{Im\_w} = \frac{C(f/f_0)m_V^x}{1 + (f/f_0)^2}$
- $f_0 = 9.07$ GHz 为湿雪有效弛豫频率

体积湿度 $m_V$ 是决定湿雪介电常数的最主要因素。
