# 冰的介电常数

在微波波段由 Ray 给出的复介电常数模型，形式与[[水的介电常数|水]]一致但参数不同。

**Ray 参数**（冰）：
- $\varepsilon_\infty = 3.168$
- $\varepsilon_s = 5.27137 + 0.0216474t - 0.00131198t^2$
- $\sigma = 1.26 \exp(-6291/(t+273))$
- $\alpha = 0.288 + 0.0052t + 0.00023t^2$
- $\lambda_s = 0.0009990288 \exp(6.6435/(t+273))$

**简化为常数模型**（< 1000 GHz）：
- $\varepsilon_{Re} = 3.15$
- $\varepsilon_{Im} = \alpha(T)/f + \beta(T)f$

**特征**：实部近似为常数（约 1.78 折射率），与频率和温度无关；虚部在 $5\times10^{-5} \sim 1\times10^{-2}$ 间变化。0°C 时衰减较 25°C 大。

理论上冰属各向异性媒质，介电常数应为张量，但文献通常未涉及。
