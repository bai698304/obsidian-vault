# 混合物等效介电常数

$\varepsilon_{eff}$：两种或多种物质组成的混合物的等效复介电常数。

假设 $n_k$ 种物质，介电常数 $\varepsilon_1, \varepsilon_2, ..., \varepsilon_{n_k}$，体积组分比 $p_1, p_2, ..., p_{n_k}$。

**六种主要模型**：

| 模型 | 适用范围 |
|------|---------|
| **Rayleigh 模型** | 通用，$\frac{\varepsilon_{eff}-1}{\varepsilon_{eff}+2} = \sum p_i \frac{\varepsilon_i-1}{\varepsilon_i+2}$ |
| **Bottcher 方程** | 通用，$\sum p_i \frac{\varepsilon_i-\varepsilon_{eff}}{\varepsilon_i+2\varepsilon_{eff}} = 0$ |
| **Berentsveig 公式** | 沙土环境 100 MHz~9 GHz 理论与实测吻合好 |
| **CRIM 模型** | 各组分粒子尺寸 < 波长时成立，$\sqrt{\varepsilon_{eff}} = \sum p_i \sqrt{\varepsilon_i}$ |
| **BH 模型** | 两相混合物（主体+客体），适用于涂覆水冰雹、含气泡冰雹、类球形水雪 |
| **Maxwell-Garnett 公式** | 含水沙尘，$\varepsilon_{eff} = \varepsilon_{dust}[1 + \frac{3p_{water}(\varepsilon_{water}-\varepsilon_{dust})/(\varepsilon_{water}+2\varepsilon_{dust})}{...}]$ |

还可将离散沉降粒子+大气背景等效为**虚拟连续媒质**，考虑粒子尺寸分布和形状的影响。
