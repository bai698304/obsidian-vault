# 开尔文-赫姆霍兹波湍流

## 定义

[[大气湍流]]三大形成机理之三，由稳定层结大气中的**切变重力波**产生。

## 产生过程

1. 稳定层结大气中，湍流通常较弱或消失
2. 但经常存在**上下层较强的风速切变**
3. → 产生**切变重力波**
4. → 风速切变足够大 → 密度被倒置 → 运动不稳定
5. → 波随振幅增大而**破碎**
6. → 破碎波的叠加 → 构成湍流
7. → 湍流形成 → 上下层混合加强 → 风速切变减弱
8. → 恢复无湍流状态 → 往复循环

## 特征

- **空间上离散**（不连续）
- **时间上间歇**（反复出现和消失）
- 常出现在：夜间稳定边界层 + 白天混合层顶
