# 随机场理论

## 基本概念

| 概念 | 研究范畴 |
|------|---------|
| **随机过程** | 时间维度的随机变化 |
| **随机场** | 空间维度的随机变化（随机过程的空间推广） |
| **空时场** | 时间+空间的联合随机变化 |

## 理想化模型

### 均匀性（空间平稳性）

| 模型            | 条件                          |          |     |
| ------------- | --------------------------- | -------- | --- |
| **均匀随机场**     | $B(r_1, r_2) = B(\Delta r)$ |          |     |
| **均匀各向同性随机场** | $B(r_1, r_2)$ = $B(         | \Delta r | )$  |

### 局部均匀性

| 模型 | 条件 |
|------|------|
| **局部均匀随机场** | $D(r_1, r_2) = D(\Delta r)$ |
| **局部均匀各向同性随机场** | $D(r_1, r_2) = D(|\Delta r|)$ |

## 实际湍流的适用性

- [[大气湍流|湍流]]电磁参数起伏一般**不能满足**均匀/各向同性条件
- 但可满足**局部均匀各向同性**假设
- → 用 [[结构函数]]（而非[[相关函数]]）来描述

## 三种研究方法

1. **统计特征法**：基于概率统计，建立 $n$ 点 $k$ 阶矩方程
2. **谱展开法**：将空间随机函数展开为不同空间谐波函数的集合
3. **Mellin 变换法**：谱展开法的变体，选用不同基函数
