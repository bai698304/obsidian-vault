# 湍流内尺度与外尺度

## 定义

| 尺度      | 符号          | 含义                                |     |
| ------- | ----------- | --------------------------------- | --- |
| **外尺度** | $l_{outer}$ | 最大涡旋尺度 ≈ [[雷诺数特征尺度  $L_{eigen}$]] |     |
| **内尺度** | $l_{inner}$ | 最小涡旋尺度（能量耗散尺度）                    |     |

## 惯性子区

$$l_{inner} \ll |\Delta r| \ll l_{outer}$$

在惯性子区内：
- [[结构函数]]遵循 **2/3 幂律**
- [[空间谱]]遵循 **-11/3 幂律**

## 典型数值

- 外尺度：与具体流动系统特征尺度相当
- 内尺度：毫米量级（由分子黏性耗散决定）
- 涡旋尺度范围：数毫米 → 数千米
