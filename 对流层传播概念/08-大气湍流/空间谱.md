# 空间谱 $\Phi(K)$

## 定义

将空间随机函数展开为具有随机复振幅的不同空间谐波函数 $A(K)e^{jK\cdot r}$ 的集合，通过分析随机复振幅的统计特性反映随机场的统计特性。

## 与相关函数的关系（均匀随机场）

$$\Phi(K) = \frac{1}{(2\pi)^3}\int \exp(-jK\cdot\Delta r)B(\Delta r)dV$$
$$B(\Delta r) = \int \exp(jK\cdot\Delta r)\Phi(K)dK$$

→ 均匀各向同性条件下可化简为一维积分。

## 与结构函数的关系（局部均匀随机场）

$$\Phi(K) = \frac{1}{16\pi^3k^2}\int \exp(-jK\cdot\Delta r)\nabla D(\Delta r)KdV$$
$$D(\Delta r) = 2\int[1 - \exp(jK\cdot\Delta r)]\Phi(K)dK$$

## 选择方法的原则

- 关注特定两点 → 用[[相关函数]]/[[结构函数]]
- 需整体了解所有点 → 用空间谱
