# 湍流五大特征

## 五大基本特征

| 特征      | 含义                    | 研究意义                   |           |
| ------- | --------------------- | ---------------------- | --------- |
| **随机性** | 非规则、混乱、不可预测           | 可用[[随机场理论]]            | 统计学]]描述 ✅ |
| **非线性** | 小扰动自发增长，很快达到扰动幅度      | 需[[K理论]]使方程闭合          |           |
| **扩散性** | 引起动量、热量及物质快速扩散        | 大气边界层输送过程              |           |
| **涡旋性** | 大小涡旋分裂、合成、拉长、旋转       | 涡旋尺度范围数毫米→数千米          |           |
| **耗散性** | 大涡旋→小涡旋能量传递→分子黏性耗散为热能 | → [[Kolmogorov湍流能谱定律]] |           |

## 判断依据

根据上述五大特征可判断湍流现象是否存在。

## 出现区域

- 大气边界层
- 积云内
- 自由大气中
