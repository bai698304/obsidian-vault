# 视线传播理论（沉降粒子环境）

沉降粒子环境中的[[视距传播]]信号由两部分构成：

**接收场** $u(r) = \langle u(r) \rangle + u_f(r)$
- $\langle u(r) \rangle$：[[平均场]]（相干场，经历衰减和相移的直达波）
- $u_f(r)$：[[起伏场]]（所有粒子散射场的叠加）

**接收强度** $I_t = I_c + I_i$
- $I_c$：[[相干强度]] — 式(6.70)
- $I_i$：[[非相干强度]] — 式(6.71)

**统计特性**：
- 起伏场相位：$0 \sim 2\pi$ 均匀分布
- 起伏场幅度：**瑞利分布** $P(A) = \frac{2A}{P_{diff}} \exp\left(-\frac{A^2}{P_{diff}}\right)$

沉降粒子环境相当于**随机多径传播效应**，参见[[沉降粒子多径信道包络PDF]]。
