# 相干强度与非相干强度比值

$I_c / I_i$ 决定接收信号包络的统计分布类型：
- $I_c/I_i \gg 1$：相干主导 → [[莱斯分布]]或确定性信道
- $I_c/I_i \sim 1$：相干与非相干可比
- $I_c/I_i \ll 1$：非相干主导 → [[起伏场|瑞利分布]]

**影响因素**：
- [[光学深度]]增大 → $I_c/I_i$ 急剧下降
- 频率升高 → $I_c/I_i$ 下降（高频散射更强）
- 降雨率增大 → $I_c/I_i$ 下降

是[[沉降粒子多径信道包络PDF]]的核心参数。
