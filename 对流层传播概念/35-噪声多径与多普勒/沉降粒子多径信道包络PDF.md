# 沉降粒子多径信道包络PDF

分析降雨、降雪、沙尘等天气现象发生时，原有[[大气多径|多径传播]]环境下的接收信号包络概率密度函数变化。

**基本多径衰落模型**（表6.13）：
- 单波无衰落 → δ 分布
- 双波简单衰落
- **瑞利分布**（无穷多径/纯非相干场）
- **莱斯分布**（主波 + 非相干场）
- **TWPD分布**（双波 + 非相干场）

**沉降粒子影响下的修正**：
- 单波模型 → 根据 $I_c/I_i$ 变为瑞利或莱斯分布
- 双波模型 → 根据 $I_c/I_i$ 变为瑞利/莱斯/TWPD
- 原有莱斯模型 → $P_{diff}$ 需同时考虑原有漫射功率衰减和新增散射功率

**$I_c/I_i$ 变化规律**：光学深度↑/频率↑/降雨率↑ → 比值急剧↓ → 非相干场主导。
