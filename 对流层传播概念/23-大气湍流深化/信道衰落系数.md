# 信道衰落系数

## 定义

描述[[大气湍流|湍流]]对[[视距传播]]信号影响程度及规律的无量纲复数系数 $h$：

$$h = \frac{U(r)}{U_0(r)} = \exp\left[\ln\left(\frac{A}{A_0}\right)\right] \exp[j(S - S_0)] = \chi \exp(j\varphi)$$

## 物理含义

- $\chi = \ln(A/A_0)$：对数幅度比 → [[幅度闪烁]]
- $\varphi = S - S_0$：相位变化 → [[相位闪烁]]
- $U_0(r)$ = 无湍流时的均值信号
- $U(r)$ = 有湍流时的瞬时信号

## 应用

任意时刻瞬时值：$U(r) = h \cdot U_0(r) = \chi \exp(j\varphi) U_0(r)$

弱起伏视距传播理论的核心就是研究 $\chi$ 和 $\varphi$ 的统计规律。

## 概率密度

地-空链路毫米波/亚毫米波系统，$\chi$ 和 $\varphi$ 均服从**高斯分布**：

$$p(\chi) = \frac{1}{\sqrt{2\pi}\sigma_\chi} \exp\left(-\frac{\chi^2}{2\sigma_\chi^2}\right)$$

$$p(\varphi) = \frac{1}{\sqrt{2\pi}\sigma_\varphi} \exp\left(-\frac{\varphi^2}{2\sigma_\varphi^2}\right)$$
