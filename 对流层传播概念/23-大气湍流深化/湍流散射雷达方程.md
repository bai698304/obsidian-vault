# 湍流散射雷达方程

## 定义

描述[[湍流非相干散射]]中收发功率关系的方程。

## 方程

$$\frac{P_R}{P_T} = \int_{V_C} \frac{\lambda^2 G_T(i) G_R(o)}{(4\pi)^3 R_1^2 R_2^2} \sigma_s(\theta_{scattering}) dV$$

其中：
- $P_R$ / $P_T$：接收 / 发射功率
- $\lambda$：信号波长
- $V_C$：收发天线**交叉公共体积空间**
- $G_T(i)$ / $G_R(o)$：发射 / 接收天线在相应方向的增益
- $R_1$ / $R_2$：发射 / 接收天线到体积元的距离
- $\sigma_s(\theta_{scattering})$：[[单位体积平均散射截面]]

## 非相干叠加前提

$V_C$ 内不同体积元的散射信号在接收点**随机相位叠加**。
