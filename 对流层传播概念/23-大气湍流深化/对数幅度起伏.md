# 对数幅度起伏

## 定义

$\ln(A/A_0)$ 的随机变化特性，用于表征[[弱起伏与强起伏|弱/强起伏]]的判据和描述[[信道衰落系数]]中的幅度分量 $\chi$。

## 与 $\chi$ 的关系

$$\chi = \ln\left(\frac{A}{A_0}\right)$$

## 弱起伏判据

对数幅度方差 $\sigma_\chi^2 < 0.2 \sim 0.5$ → 弱起伏；否则 → 强起伏。

## $\chi$ 的概率分布

$$p(\chi) = \frac{1}{\sqrt{2\pi}\sigma_\chi} \exp\left(-\frac{\chi^2}{2\sigma_\chi^2}\right)$$

## 注意

- $\sigma_\chi$（对数振幅标准差）≠ [[闪烁强度|\sigma_A]]（幅度标准差）
- 阅读文献时需区分是 $\sigma_\chi$ 还是 $\sigma_A$
