# 微分散射截面

$$\sigma_d(o,i) = \lim_{R\to\infty} \left(\frac{R^2 |S_s|}{|S_i|}\right) = |f(o,i)|^2 = \frac{dP_s/d\Omega}{|S_i|}$$

**量纲**：面积 / 单位立体角

度量粒子将入射波散射到 $o$ 方向单位立体角的能力，仅是散射角 $\theta_s$（$o$ 与 $i$ 的夹角）的函数。

对于雷达应用：
- **双站雷达截面** $\sigma_{bi} = 4\pi \sigma_d(o,i)$
- **后向散射截面** $\sigma_b = 4\pi \sigma_d(-i,i)$ → [[雷达方程]]
