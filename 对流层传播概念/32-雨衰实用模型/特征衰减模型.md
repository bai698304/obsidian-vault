# 特征衰减模型（降雨）

Olsen 等基于负指数雨滴谱和 Mie 级数展开得到的[[衰减率]]工程模型：

$$\gamma_{dB/km} = a_r [R(1)]^{b_r}$$

- $R(1)$：[[1分钟累积降雨率]]
- $a_r, b_r$：[[降雨特征衰减回归系数 a和b]]

**极化修正**（$a_r, b_r$ 与极化有关）：
$$a_r = \frac{1}{2}[a_H + a_V + (a_H - a_V)\cos^2\theta \cos 2\tau]$$

- $\tau = 0°$ 水平极化，$\tau = 90°$ 垂直极化，$\tau = 45°$ 圆极化
- $\theta$：链路仰角

[[ITU-R P.838]] 给出了 1~1000 GHz 的推荐参数表。

不同雨滴谱（Gamma、L-P、M-P、对数正态）对应不同参数 → IEEE 802.16cc-99/24。
