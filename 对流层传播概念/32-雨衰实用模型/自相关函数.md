# 自相关函数（ACF）

$$\mathrm{ACF}(s) = \frac{\mathrm{Cov}(x_t, x_{t+s})}{\sqrt{D(x_t) D(x_{t+s})}}$$

对于平稳测量序列 $A_i$：
$$c(i) = \frac{1}{N} \sum_{t=1}^{N-i} (A_t - \bar{A})(A_{t+i} - \bar{A})$$
$$\mathrm{ACF}(s) = \rho(s) = \frac{c(s)}{c(0)}$$

**平稳性检验**：
- ACF 快速衰减至 0 → **平稳** → 可直接用 [[雨衰实时预报ARIMA|ARMA]]
- ACF 衰减缓慢 → **非平稳** → 需差分后用 [[雨衰实时预报ARIMA|ARIMA]]

由 SPSS 等时间序列软件直接计算。参见 [[偏相关函数]]。
