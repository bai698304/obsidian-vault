# 路径缩减因子

[[降雨等效路径模型]]中将几何路径长度转换为[[等效路径长度]]的缩减系数。

**ITU-R 地面链路** 路径缩减因子：
$$r_F = \frac{1}{1 + \frac{L_0}{35 e^{-0.015 R_{0.01}}}} \quad (R_{0.01} \leqslant 100\mathrm{mm/h})$$

物理意义：由于[[雨胞]]尺度有限，长路径上并非处处都是强降雨，等效路径 < 几何路径。

对于地-空链路，ITU-R 分别定义了**水平路径校正因子** $r_{H0.01}$ 和**垂直路径缩减因子** $r_{V0.01}$。
