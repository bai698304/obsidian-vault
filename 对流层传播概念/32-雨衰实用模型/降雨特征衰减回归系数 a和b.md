# 降雨特征衰减回归系数 a 和 b

[[特征衰减模型]] $\gamma = a_r R(1)^{b_r}$ 中的回归系数。

- $a_H, b_H$：水平极化参数
- $a_V, b_V$：垂直极化参数

**确定方法**：
1. 查表：[[ITU-R P.838]] 提供 1~1000 GHz 推荐值
2. 插值：$a_r$ 对数插值，$b_r$ 线性插值
3. 解析公式：Olsen 等给出的分段解析式
4. **实测回归**：利用[[雨滴谱仪测量确定a和b|雨滴谱仪数据]]通过 $\log(\gamma) = \log(a_r) + b_r \log[R(1)]$ 线性回归

**地区差异性**：不同地区的 $a_r, b_r$ 差异显著。以西安 vs ITU-R 为例，50 GHz 时特征衰减差可达 6 dB。
