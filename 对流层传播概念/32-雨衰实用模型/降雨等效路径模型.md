# 降雨等效路径模型

将穿过[[雨胞|非均匀雨区]]的几何路径折合为等效路径 $L_{e\_km}$ 的方法。

**地面链路模型**：ITU-R / Moupfouma / CETUC / Garcia / China / Brazil 等
- $L_{e\_km} = L_0 \times r_F$，$r_F$ 为[[路径缩减因子]]

**地-空链路模型**：20+ 种（Assis-Einloft、Crane、EXCELL、ITU-R P.618 系列、SAM、DAH、China 等）

**最重要的两个模型**：
- [[ITU-R等效路径模型]]（ITU-R P.618-9）：六步法，同时考虑水平和垂直不均匀
- [[China等效路径模型]]（赵振维、林乐科）：基于中国实测数据
