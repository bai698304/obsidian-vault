# 降雨率水平空间分布模型

描述[[雨胞]]中心向外降雨率随距离变化的数学模型。

| 模型 | 表达式/特点 |
|------|-----------|
| **双层圆柱模型** | 内层半径 $D_{INT\_C} = 2.2(100/R_{INT\_C})^{0.4}$ km，外区直径约 33 km |
| **指数模型** | $R(l) = R_{peak} \exp(-4l/l_{radius})$ |
| **高斯模型** | $R(l) = R_{peak} \exp[-\frac{1}{2}(3l/0.8l_{radius})^2]$ |
| **EXCELL模型** | $R(x,y) = R_{peak} \exp[-(x^2/a_x^2 + y^2/a_y^2)^{1/2}]$ |
| **HYEXCEL模型** | EXCELL 扩展，含转折点降雨率的两段式分布 |

降雨率垂直空间分布尚无定量化物理模型。
