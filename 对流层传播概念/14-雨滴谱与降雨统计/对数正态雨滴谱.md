# 对数正态雨滴谱

满足对数正态分布的雨滴粒子空间分布模型。

表达式：
$$N(D) = \frac{N_T}{\sigma D \sqrt{2\pi}} \exp\left[-\frac{1}{2}\left(\frac{\ln D - \mu}{\sigma}\right)^2\right]$$

其中：
- $N_T = a_1 R^{b_1}$
- $\mu = a_2 + b_2 \ln R$
- $\sigma^2 = a_3 + b_3 \ln R$

参数取值因降雨类型（Drizzle / Widespread / Shower / Thunderstorm）和气候区（A. Maitra / Ajayi-Olsen）而异。

常用于描述**热带雨林气候**的雨滴尺寸分布特性。
