# 永久性气体（不可变成分）

## 定义

在[[干洁大气]]中滞留时间在 **1～10³ 年**范围内，且各成分所占比例**大致保持不变**的气体。

## 成员

| 气体 | 容积含量 |
|------|---------|
| N₂ | 78.084% |
| O₂ | 20.948% |
| Ar | 0.934% |
| CO₂ | 0.033% |
| Ne | 1.8×10⁻³ |
| He | 5×10⁻⁴ |
| Kr | 1×10⁻⁴ |
| Xe | 9×10⁻⁶ |
