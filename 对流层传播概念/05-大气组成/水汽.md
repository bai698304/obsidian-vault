# 水汽

## 定义

由水蒸发而成的气体。[[对流层]]晴空大气实际上是[[干洁大气]]和水汽的混合体。

## 关键数据

| 特征 | 数值 |
|------|------|
| 在大气中占比 | 0.1%～0.4% |
| 在对流层中的集中度 | 90% 以上 |
| 主要来源 | 海洋表面蒸发 |

## 重要性

- 虽然含量不多，但**能成云致雨**
- 是大气中**最活跃**的成分
- 是[[暖云降雨]]和[[冷云降雨]]的必备条件 → [[云雾形成条件]]
- 对对流层中电波传播有**非常重要的影响**
- [[常折射指数 N₀]]的湿项 $N_{0\_wet}$ 全年变化约 20～100 N 单位，时空变化主因
- [[水汽30条谱线]]是[[吸收|大气吸收]]计算的重要组成部分
- 水汽凝结形成[[云雾滴]] → [[含水量]]决定云雾对电磁波的衰减
