# 凭借效应与限制效应

## 定义

对流层环境对无线系统有**两方面**的影响作用：

### "凭借"效应（正面）

对流层环境成为实现**特定传播模式**的"凭借"作用：
- **[[大气湍流|湍流大气结构]]**：实现[[对流层散射传播|散射超视距通信]]
- **[[悬空波导|悬空波导结构]]**：实现超视距探测/通信
- **[[蒸发波导|蒸发波导结构]]**：海面超视距传播
- **[[散射|沉降粒子]]、[[大气湍流|湍流结构]]**：为毫米波地-空 MIMO 提供丰富散射体

### "限制"效应（负面）

对流层环境对无线电信号传播的**"限制"**作用和对信息的**"污染"**作用：
- [[衰减]] → 信号强度下降
- [[闪烁]] → 幅度相位随机起伏
- [[去极化]] → 极化失配
- [[大气噪声]] → 信噪比降低
- [[多普勒频移]] → 相干检测困难
- [[附加时延]] → 时序同步问题

## 研究目标

> 有效克服对流层环境对无线系统的**负面影响**，有效利用其**正面影响**。

使系统的工作性能与对流层环境的信道特性达到**良好匹配**。
