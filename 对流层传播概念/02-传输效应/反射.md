# 反射

## 定义

分层大气中，电磁波在大气折射**骤变==边界**==处的一种特有现象。发生在两种不同运动规律气流的交界处。

## 物理机理

| 特性 | 说明 |
|------|------|
| 发生条件 | 两种不同运动规律气流的交界处（大气边界层） |
| 传输模型 | 菲涅尔反射理论模型 |
| 折射分量 | 部分信号按 Snell 折射定律发生折射传播 |
| 关键特征 | 折射率处于**"不连续"**状态 |

## 与散射的对比

| | [[散射]] | [[反射]] |
|----|---------|---------|
| 对象 | 不均匀体 | 层状结构 |
| 电流排列 | 不规则 → 类似偶极子 | 规则阵列 → 类似阵列天线 |
| 辐射方向 | 向四面八方 | 向符合 Snell 定律的方向 |

## 对比

- **[[折射]]**：折射率连续，轨迹弯曲
- **反射**：折射率骤变（不连续），菲涅尔反射
