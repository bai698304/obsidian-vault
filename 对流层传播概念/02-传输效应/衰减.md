# 衰减

## 定义

电磁波在传播过程中**能量减少**的现象。

## 两种衰减机理

| 类型 | 英文 | 机理 | 能量转换 |
|------|------|------|---------|
| 自由空间扩散衰减 | Free-Space Spreading Loss | 仅因波阵面扩散导致单位面积能量减小 | ❌ 无 |
| 媒质衰减 | Medium Attenuation | [[吸收]] + [[散射]] | ✅ 有（电磁能→内能） |

### 吸收效应导致衰减

[[吸收|电磁波能量 → 传播媒质内能]]，伴随能量转换过程。

### 散射效应导致衰减

[[散射]]使电磁波能量改变传播方向，在接收方向上能量减少。

## 主导判据（沉降粒子环境）

| 条件 | 主导机理 |
|------|---------|
| 粒子线度 ≪ 波长 | [[吸收]]主导 |
| 粒子线度 ≫ 波长 | [[散射]]主导 |

## 沉降粒子衰减的关键因素

- [[雨滴尺寸分布谱]] + [[降雨率]] → 降雨衰减量
- [[云雾滴谱]] + [[含水量]] → 云雾衰减量
- [[1分钟累积降雨率]]是电波传播衰减计算的标准参数
- 粒子介电特性（[[水的介电常数]]、[[冰的介电常数]]）决定吸收效率

## 晴空大气衰减

- [[大气吸收衰减率]] → [[大气吸收衰减]] → [[地空链路大气吸收]]
- [[Liebe模型]] / [[ITU-R大气吸收模型]] / [[CCIR吸收衰减模型]]
- [[氧气吸收衰减率]] / [[水汽吸收衰减率]] / [[大气窗口频率]]

| 频段 | 主导衰减机理 |
|------|------------|
| 无线电波段 | [[吸收]]（晴空大气） |
| 毫米波 | [[吸收]] + [[散射]]（沉降粒子影响显著） |
| 光波段 | [[吸收]] + [[散射]]共同决定（[[云雾滴|云雾]]影响最严重） |

## 统一称呼

[[消光效应]] = 吸收效应 + 散射效应

## 系统影响

衰减是[[凭借效应与限制效应|限制效应]]的典型体现 → 恶化[[无线系统性能]]
