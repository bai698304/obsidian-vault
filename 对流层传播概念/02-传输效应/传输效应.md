# 传输效应

## 定义

对流层环境对电磁波产生的各种传播和散射效应的总称。

## 效应清单

| 效应          | 环境            | 核心笔记      |
| ----------- | ------------- | --------- |
| [[折射]]      | 晴空大气连续分层      | 轨迹弯曲      |
| [[反射]]      | 大气折射骤变边界      | 菲涅尔+Snell |
| [[散射]]      | 湍流不均匀体 + 沉降粒子 | 二次辐射      |
| [[吸收]]      | 气体分子 + 粒子     | 能级跃迁      |
| [[衰减]]      | 吸收+散射导致       | 能量减少      |
| [[相移]]      | 不同媒质波长差异      | 相位滞后      |
| [[去极化]]     | 晴空大气 + 沉降粒子   | 极化状态变化    |
| [[大气噪声]]    | 气体分子/粒子       | 自发热辐射     |
| [[对流层间接干扰]] | 散射/反射/折射/去极化  | 间接引起      |
| [[多普勒频移]]   | 对流层环境         | 频率变化      |
| [[闪烁]]      | 湍流            | 幅度相位随机起伏  |
| [[附加时延]]    | 对流层           | 额外时间延迟    |

## 与系统性能的关系

所有传输效应最终体现在[[无线系统性能]]上，包含：
- **[[凭借效应与限制效应|凭借效应]]**：促成特定传播模式（散射超视距/波导超视距）
- **[[凭借效应与限制效应|限制效应]]**：恶化系统容量/误码率/稳定度/发现概率/定位精度等

## 频率依赖性

不同传输效应在不同频段表现程度不同：
- 无线电波段：[[衰减]]主要由[[吸收]]引起
- 光波段：[[衰减]]由[[吸收]] + [[散射]]共同决定
- ==粒子线度== ≪ 波长 → 吸收主导
- 粒子线度 ≫ 波长 → 散射主导
