# 散射

## 定义

电磁波==辐射==到大气折射指数"不均匀体"或沉降粒子上时，电磁波能量改变了原来的传播方向，朝**不同方向**传播的现象。

## 两种散射

### 1. 晴空大气散射

- **物理机理**：[[大气湍流|湍流运动]]使局部大气折射指数出现起伏 → 形成"不均匀体"
- 不均匀体感应电流 → 像"天线"一样成为**二次辐射体**
- 感应电流结构不能呈现规则阵列排列
- → 二次辐射类似等效**偶极子** → 能量向**四面八方**辐射

### 2. 沉降粒子散射

- [[雨滴直径|雨滴]]、[[云雾滴|云滴/雾滴]]、[[冰晶]]、[[雪|雪花]]、[[冰雹]]、[[沙尘暴|沙尘]]等沉降粒子作为二次辐射体
- 粒子感应电流 → 二次辐射
- 散射强度与[[粒子线度范围|粒子线度]]密切相关：线度与波长接近时散射最显著
- 大粒子（冰雹/大雨滴）散射效应远超吸收效应

## 与反射的微妙关系

| 散射 | 反射 |
|------|------|
| 不均匀体 | 层状结构 |
| 感应电流不规则排列 | 感应电流规则阵列排列 |
| 二次辐射类似偶极子 | 二次辐射类似阵列天线 |
| 能量向四面八方 | 能量向 Snell 反射定律方向 |

## 散射与衰减

- 散射也是[[衰减]]的形成机理之一
- [[消光效应]] = 吸收 + 散射
- 粒子线度 ≪ 波长 → [[吸收]]主导衰减
- [[粒子线度]] ≫ 波长 →[[散射]] 主导衰减
