# 无线系统性能

## 定义

对流层[[传输效应]]最终影响的无线系统性能指标。

## 受影响的性能指标

| 指标 | 英文 | 影响机理 |
|------|------|---------|
| 容量 | Capacity | [[衰减]]降低 SINR |
| 误码率 | BER | [[闪烁]] + [[大气噪声]] + [[去极化]] |
| 稳定性 | Stability | [[闪烁]]随机起伏 |
| 可靠度 | Reliability | 所有传输效应的统计特性 |
| 发现概率 | Detection Probability | [[衰减]] + [[大气噪声]] |
| 虚警概率 | False Alarm Probability | [[对流层间接干扰]] + [[大气噪声]] |
| 定位制导精度 | Positioning Accuracy | [[折射]]引起的误差 |
| 分辨率 | Resolution | [[衰减]] + [[去极化]] + [[多普勒频移]] |

## 研究目标

使系统工作性能与[[凭借效应与限制效应|对流层信道特性]]达到**良好匹配**。
