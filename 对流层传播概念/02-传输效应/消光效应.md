# 消光效应

## 定义

[[吸收]]效应和[[散射]]效应的**统一称呼**。

## 说明

由粒子对电磁波的散射理论可知，可以把吸收效应和散射效应统称为消光效应。
当[[粒子线度范围|粒子线度]]与电磁波波长接近时消光效应最显著（对毫米波/光波影响 > 厘米波）。
消光效应的计算依赖于粒子的[[雨的介电常数|介电特性]]和[[雨滴尺寸分布谱|尺寸分布谱]]。

## 关系

```
消光效应 = 吸收效应 + 散射效应
         = 衰减（有能量转换部分）
```

## 相关概念

- [[衰减]]：消光效应 + 自由空间扩散衰减 = 总衰减
- [[吸收]]：消光的吸收分量
- [[散射]]：消光的散射分量
