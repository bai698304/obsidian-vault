# 波导传输损耗

## 定义

[[大气波导]]环境中的总传输损耗 $L_D$：

$$L_D = L_b + L_m$$

- $L_b$：**基本传输损耗**（扩散效应 + 传播模式导致的接收功率减小，无能量转化）
- $L_m$：**媒质吸收损耗**（电磁能向内能转化）

## 媒质吸收损耗

$$L_m = \int_0^S \gamma_a(s) ds$$

沿射线路径积分[[大气吸收衰减率]]。

## 基本传输损耗的三种计算方法

| 理论 | 核心公式 |
|------|----------|
| [[几何光学理论]] | $L_b = 20\lg \frac{|E(s_b)|}{|E(s_a)|}$ |
| [[波导模理论]] | $L_b = 32.45 + 20\lg f + 20\lg r - \text{ECMS/EIMS}$ |
| [[抛物型方程法]] | $L_b = 32.45 + 20\lg f + 10\lg x - 20\lg|q|$ |
