# 边界条件

## 定义

求解[[抛物型方程法|抛物型方程]]必须处理的三类问题之一，包括上边界吸收条件和下边界物理条件。

## 上边界：Sommerfeld 辐射条件

电磁波到达上边界时被**完全吸收**，不向计算区域内反射或透射。通过窗函数实现：
- Hanning Window
- Cosine-taper (Tukey) 窗函数

## 下边界：第三类混合边界条件

$$\beta \frac{\partial q(x,z)}{\partial z}\bigg|_{z=0} + \alpha q(x,z)\bigg|_{z=0} = 0$$

| 下边界类型 | $\alpha$, $\beta$ 取值 |
|-----------|----------------------|
| 光滑理想导体（水平极化） | $\alpha=1, \beta=0$ |
| 光滑理想导体（垂直极化） | $\alpha=0, \beta=1$ |
| 光滑良导体 | $\alpha$ 含介质参数，$\beta=1$ |
| 光滑阻抗边界 | $\alpha = jk\sin\theta(1-\Gamma)/(1+\Gamma)$, $\beta=1$ |
