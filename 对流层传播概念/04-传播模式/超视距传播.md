# 超视距传播

## 定义

收发天线间由于存在地形、地物等障碍物而处于**"不可见"**状态，电磁信号必须经过**中继接力**才能到达接收天线的传播模式。

## "不可见"的严格定义

障碍物遮挡了收发天线之间第一[[菲涅尔区]]的 **60% 以上**区域。

## 本质

超视距传播 = 几个[[视距传播]]模式的**组合传播**结果。

## 实现途径

| 途径 | 中继接力方式 |
|------|------------|
| [[对流层散射传播]] | 湍流不均匀体散射 |
| [[大气波导传播]] | 大气波导折射陷获 |
| 卫星中继 | 通信卫星转发 |
| 电离层折射 | 电离层反射 |
| 流星余迹 | 流星尾迹散射 |
| 地面绕射 | 地形绕射 |

## 散射传播三理论

| 理论 | 核心观点 |
|------|---------|
| **湍流不相干散射理论** | 对流层不均匀体的散射作用 |
| **稳定层相干反射理论** | 介电常数随高度稳定非线性分布 |
| **多模理论（不规则层非相干反射）** | 水平和垂直方向上不规则突变层的反射 |
