# 修正范弗莱克-威斯科福模型

## 定义

由 Leibe 在范弗莱克-威斯科福模型基础上修正的[[谱线形状因子]]模型，被 **ITU-R** 采用于计算晴空大气[[吸收|吸收衰减]]。

## 四种谱线模型对比

| 模型 | 地位 |
|------|------|
| 洛伦兹模型 | 最早/最简单 |
| 范弗莱克-威斯科福模型 | 改进版 |
| 格洛斯模型 | 另一种形式 |
| **修正范弗莱克-威斯科福** ✅ | ITU-R 标准 |

## 形状因子公式

$$F_{i\_Re} = \frac{1}{f_i}\left[\frac{f_i^2-\Delta f^2 - f(1-\delta\Delta f)}{(f_i-f)^2+\Delta f^2} + \frac{f_i^2-\Delta f^2 + f(1-\delta\Delta f)}{(f_i+f)^2+\Delta f^2} + 2\right]$$
$$F_{i\_Im} = \frac{1}{f_i}\left[\frac{\Delta f - \delta(f_i-f)}{(f_i-f)^2+\Delta f^2} + \frac{\Delta f + \delta(f_i-f)}{(f_i+f)^2+\Delta f^2}\right]$$
