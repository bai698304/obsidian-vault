# 折射指数 $N$

## 定义

为方便估计而定义的电磁特性参数，与[[折射率 n]]的关系：

$$N = (n-1) \times 10^6$$

## 分量构成

$$N = N_0 + N_{Re}(f) + jN_{Im}(f)$$

| 分量                 | 含义             | 频率依赖 |     |
| ------------------ | -------------- | ---- | --- |
| [[常折射指数 N₀]]       | 基础项，与频率和分子共振无关 | ❌ 无  |     |
| [[色散项$N_{Re}(f)$]] | 色散项实部（与频率有关）   | ✅ 有  |     |
| $N_{Im}(f)$        | 色散项虚部（与吸收有关）   | ✅ 有  |     |

## 物理基础

$N_0$ 由[[气象三要素]]决定：

$$N_0 = 77.6\frac{P}{T} + 3.73 \times 10^5 \frac{e}{T^2}$$

- 第一项：干项 $N_{0\_dry}$（年变化 ≈ 20 N 单位）
- 第二项：湿项 $N_{0\_wet}$（年变化 ≈ 20-100 N 单位）

## 色散项

$N_{Re}(f)$ 和 $N_{Im}(f)$ 包含：
- [[氧气48条谱线|氧气共振项]]（48 条谱线）
- [[水汽30条谱线|水汽共振项]]（30 条谱线）
- 非共振项（O₂/N₂ + 水汽）

## 时空分布

- 以**球面分层结构**形态为主
- 地面 2-3 km 以下受水汽主导（变化大）
- 4-5 km 以上由干空气决定（变化小）
- 8-9 km 全球年均值 ≈ 105 N 单位
