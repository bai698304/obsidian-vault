# 氧气 48 条谱线

## 概述

O₂ 分子具有 **48 条共振谱线**，频率范围约 50～834 GHz，用于计算[[色散项|$N_{O\_Re}$ 和 $N_{O\_Im}$]]。

## 计算贡献

$$N_{O\_Re}(f) = \sum_{i=1}^{48} S_i F_{i\_Re}$$
$$N_{O\_Im}(f) = \sum_{i=1}^{48} S_i F_{i\_Im}$$

## 参数表

每条谱线对应 6 个参数：

| 参数 | 含义 |
|------|------|
| $f_i$ (GHz) | 谱线频率 |
| $a_1$ | [[谱线强度]]参数 1 |
| $a_2$ | [[谱线强度]]参数 2 |
| $a_3$ | [[谱线宽度与重叠系数|谱线宽度]]参数 1 |
| $a_4$ | [[谱线宽度与重叠系数|谱线宽度]]参数 2 |
| $a_5$ | [[谱线宽度与重叠系数|重叠系数]]参数 1 |
| $a_6$ | [[谱线宽度与重叠系数|重叠系数]]参数 2 |

> 详细参数取值见原书表 2.4。
