# 光波段折射指数

## 适用范围

当频率为光频率范围时，$N = N_0 + N_{Re}(f) + jN_{Im}(f)$ 依然成立，但与无线电波段有以下差异。

## 与无线电波段的差异

| 特征 | 无线电波段 | 光波段 |
|------|----------|--------|
| $N_0$ 与频率关系 | 无关 | **有关** |
| 共振谱线密度 | 78 条（O₂+H₂O） | 远比无线电波段密集 |
| 大气湍流影响 | 有 | **更明显** |
| 谱线强度/形状因子公式 | 见 2.2.1 节公式 | 不能使用无线电波段公式 |

## 光波段 $N_0$ 公式

$$N_0 = C_\lambda\frac{P}{T}\left(1 - 0.132\frac{e}{P}\right)$$

$C_\lambda$ 是与波长有关的参数，可见光波段湿度影响可忽略。

干燥空气近似（可见光区）：
$$N_0 = \left(C_1 + \frac{C_2}{\lambda^{C_3}}\right)\frac{P}{T}$$

参数取值因文献而异。
