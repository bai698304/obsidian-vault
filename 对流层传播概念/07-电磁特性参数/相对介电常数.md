# 相对介电常数 $\varepsilon_r$

## 定义

描述媒质**电磁特性**的基本物理量。从一般意义上讲为复数：

$$\varepsilon_r = \varepsilon_{r\_Re} + j\varepsilon_{r\_Im}$$

## 与其他参数的关系

| 关系 | 公式 |
|------|------|
| 介电常数 | $\varepsilon = \varepsilon_0\varepsilon_r$（$\varepsilon_0$ 为真空介电常数） |
| 折射率 | $n = \sqrt{\varepsilon_r}$ |
| 磁导率 | $\mu = \mu_0\mu_r$（对流层 ≈ $\mu_0$，偏差仅 $10^{-3} \sim 10^{-5}$） |

## 在湍流中的起伏

[[大气湍流|湍流大气]]中 $\varepsilon_r$ 随空间呈现随机分布：

$$\varepsilon_r(r, t) = \langle\varepsilon_r(t)\rangle + \Delta\varepsilon_r(r, t)$$

$\Delta\varepsilon_r$ 的起伏特性由[[相关函数]]、[[结构函数]]和[[空间谱]]描述。
