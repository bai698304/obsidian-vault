# 波导传播四条件

## 四个必要条件

边界层大气中的电磁波若要形成[[大气波导传播|波导传播]]，必须**同时满足**：

| 条件 | 说明 |
|------|------|
| **① 存在波导** | 近地层或边界层某一高度处存在 $dM/dh < 0$ 的大气层结 |
| **② 波长条件** | 电磁波波长 < [[陷获波长|最大陷获波长 $\lambda_{max}$]] |
| **③ 位置条件** | 发射源位于[[大气波导|波导层]]内；对[[悬空波导]]，发射源距波导底不远+波导非常强时也可 |
| **④ 角度条件** | 发射仰角 < [[穿透角|临界仰角 $\theta_c$]] |
