# 波导形成条件

## 数学条件

[[大气波导]]形成的充要条件：

$$\frac{dN_{Re}}{dh} < -0.157 \text{ m}^{-1}$$

或等价地（用[[修正折射指数 M]]）：

$$\frac{dM}{dh} < 0$$

## 气象条件

当以下大气现象出现时，形成波导的**概率很大**：

| 现象 | 阈值 |
|------|------|
| [[逆温]]（温度逆增） | 梯度 > **8.5℃/100m** |
| [[湿降]]（湿度锐减） | 梯度 > **2.9 hPa/100m** |
| 两者联合 | 各自不满足阈值时也可能形成 |

## 产生大气过程

- 空气对流
- 下沉逆温
- 辐射冷却
- 水汽蒸发
- 锋面过程
