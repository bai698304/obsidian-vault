# RASS（无线电声波探测系统）

## 全称

Radio Acoustic Sounding System

## 原理

将无线电波与声波相结合：
1. 声波与媒质/目标相互作用
2. → 媒质/目标的物理、电磁特性发生变化
3. → 影响无线电波的传播、散射特性

## 应用领域

| 应用 | 说明 |
|------|------|
| 大气物理特性参数反演 | 温度剖面、风廓线、湍流 |
| 材料应力精细结构分析 | — |
| 目标探测 | 地雷/矿物/肿瘤探测 |
| 对流层探测 | 对流层低层大气温度/风廓线 |

## 发展历史

- 1950s：理论机理分析论证
- 第一台测试大气温度剖面的 RASS → **斯坦福大学**
- 1990s 末：推广至更广泛应用领域
