# 阵列干涉因子 $S(\theta, \phi)$

## 定义

决定[[相干声波扰动|声波干涉]]空间分布的数学因子。干涉区域空间位置 $r$ 处声压分布：

$$p(r) = p_0(r) |S|$$

$p_0(r)$：单个声源在 $r$ 处的声压，$S$：相干声源阵列因子。

## 点源平面阵

$$S(\theta, \phi) = \frac{\sin\left(N_x\frac{\pi b_x\sin\theta\cos\phi}{\lambda} + \frac{\beta_x}{2}\right)}{\sin\left(\frac{\pi b_x\sin\theta\cos\phi}{\lambda} + \frac{\beta_x}{2}\right)} \cdot \frac{\sin\left(N_y\frac{\pi b_y\sin\theta\sin\phi}{\lambda} + \frac{\beta_y}{2}\right)}{\sin\left(\frac{\pi b_y\sin\theta\sin\phi}{\lambda} + \frac{\beta_y}{2}\right)}$$

## 线源阵列

$$S(\theta) = \frac{\sin\left(N\frac{b\sin\theta}{\lambda} + \beta\right)}{\sin\left(\frac{b\sin\theta}{\lambda} + \beta\right)}$$

| 参数 | 含义 |
|------|------|
| $N_x, N_y$ | x/y 方向阵元数 |
| $b_x, b_y$ | x/y 方向阵元间距 |
| $N, b$ | 线源个数及间距 |
| $\lambda$ | 声波波长 |
| $\beta_x, \beta_y, \beta$ | 相邻声源间初始振动相位差 |

## 控制作用

通过调整上述参数 → 控制[[人工不均匀体阵列]]的形状/尺寸/分布。
