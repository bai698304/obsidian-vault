# 点源平面阵与线源平面阵

## 点源平面阵

均匀排列的**二维**点声源阵列（$N_x \times N_y$ 个点源）。

[[阵列干涉因子]]：

$$S(\theta, \phi) = \frac{\sin(N_x\frac{\pi b_x\sin\theta\cos\phi}{\lambda} + \frac{\beta_x}{2})}{\sin(\frac{\pi b_x\sin\theta\cos\phi}{\lambda} + \frac{\beta_x}{2})} \cdot \frac{\sin(N_y\frac{\pi b_y\sin\theta\sin\phi}{\lambda} + \frac{\beta_y}{2})}{\sin(\frac{\pi b_y\sin\theta\sin\phi}{\lambda} + \frac{\beta_y}{2})}$$

### 不均匀体尺度

$$l_\theta = r_c\Delta\theta \quad\text{或}\quad l_\theta = r_c\frac{\Delta\theta}{2}$$
$$l_\phi = r_c\tan\theta\Delta\phi \quad\text{或}\quad l_\phi = r_c\tan\theta\frac{\Delta\phi}{2}$$

## 线源平面阵

均匀排列的**一维**线声源阵列（N 个线源平行排列）。

[[阵列干涉因子]]：

$$S(\theta) = \frac{\sin(N\frac{b\sin\theta}{\lambda} + \beta)}{\sin(\frac{b\sin\theta}{\lambda} + \beta)}$$

### 不均匀体尺度

- **长度**：与线源长度一致
- **宽度**：$l_\theta = r_c\Delta\theta$（相邻干涉相消位置距离）

## 控制参数总结

| 参数 | 含义 |
|------|------|
| $N_x, N_y, N$ | 阵元数 |
| $b_x, b_y, b$ | 阵元间距 |
| $\beta_x, \beta_y, \beta$ | 相邻声源初始相位差 |
| $\lambda$ | 声波波长（由频率决定） |
| $r_c$ | 干扰区域中心到阵列声源中心的距离 |
