# 对流层人工变态

## 定义

在一定条件下**改变和控制**对流层环境的物理、电磁特性，进而改变和控制环境媒质信道对电波传播的**"凭借"和"限制"**效应，使信道对系统的影响朝**预想方向发展**。

## 核心思路

从被动适应 → 主动控制：
- 自然环境下 → 电波传播效应不可控
- 人工变态 → 定向影响电波传播特性

## 技术现状

| 领域 | 技术示例 |
|------|---------|
| 人工海洋环境 | 海啸风暴/巨浪/海雾 |
| 人工气象环境 | 气象伪装/清障/侵袭/干扰 |
| 人工大气 | 云雾/闪电/电磁脉冲/降雨/造旱 |
| 人工电离层 | 电离层加热/人造极光/臭氧层空洞 |
| 人工太阳辐射 | 阳光聚焦/阳光板 |

## "十一五"863计划探索

- 方向：声波干扰对流层 → 增强散射通信能力
- 成果：理论证明了[[相干声波扰动]]产生[[人工不均匀体阵列]]的可行性
- 结论：理论可行，未被批准立项，研究被搁置

## 应用前景

- 散射超视距"凭借"条件**人为可控制**
- 模拟大气[[闪烁]] → 研究对抗算法
- [[电子对抗]]新手段
- 散射通信能力增强
