# 对流层顶特征参数

## 定义

用于分析[[对流层顶]]气候特征及其运动规律的描述参数。

## 七类特征参数

| 参数 | 定义 |
|------|------|
| **[[对流层顶中心高度]]** | 对流层向平流层过渡层内，以地面主要热源和以平流层气体主要热源的交界面距地表高度 |
| **[[对流层顶下限高度]]** | 大气三要素取值或梯度相对于对流层内发生不连续的最低位置距地表高度 |
| **[[对流层顶上限高度]]** | 大气三要素取值或梯度相对于平流层内发生不连续的最高位置距地表高度 |
| **[[对流层顶厚度]]** | 上下限高度位置之间的间距 |
| **内部温度/压强/湿度场** | 对流层顶薄层内的温度、压强、湿度取值及变化规律 |
| **内部风场参数** | 对流层顶薄层内大气环流及其他模式运动的特征参数 |
| **特征参数变化规律** | 上述参数随高度、经纬度或时间的变化规律 |

> 目前尚无权威文献针对上述参数给出系统数据。
