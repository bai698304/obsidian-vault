# 二维离散型随机变量

## 定义

如果二维随机变量 $(X, Y)$ 的可能取值是有限对值或可列无限对值，则称 $(X, Y)$ 为**二维离散型随机变量**。

## 联合分布律

称

$$
p_{ij} = P\{X = x_i, Y = y_j\}, \quad i, j = 1, 2, \dots
$$

为 $(X, Y)$ 的**分布律**或随机变量 $X$ 和 $Y$ 的**联合分布律**，记为 $(X, Y) \sim p_{ij}$。

## 充要条件

1. $p_{ij} \geqslant 0$
2. $\sum_{i=1}^{\infty} \sum_{j=1}^{\infty} p_{ij} = 1$（归一性）

## 与边缘分布、条件分布的关系

- [[二维离散型随机变量的边缘分布]]：$p_{i\cdot} = \sum_j p_{ij}$，$p_{\cdot j} = \sum_i p_{ij}$
- [[二维离散型随机变量的条件分布]]：$P\{X = x_i \mid Y = y_j\} = \frac{p_{ij}}{p_{\cdot j}}$（条件分布 = 联合分布 / 边缘分布）
