# 二维离散型随机变量的边缘分布

## 定义

设 $(X, Y) \sim p_{ij}$，则 $X$ 和 $Y$ 的**边缘分布**分别为

$$
p_{i\cdot} = P\{X = x_i\} = \sum_{j=1}^{\infty} P\{X = x_i, Y = y_j\} = \sum_{j=1}^{\infty} p_{ij} \quad (i = 1, 2, \dots)
$$

$$
p_{\cdot j} = P\{Y = y_j\} = \sum_{i=1}^{\infty} P\{X = x_i, Y = y_j\} = \sum_{i=1}^{\infty} p_{ij} \quad (j = 1, 2, \dots)
$$

## 口诀

- $X$ 的边缘分布：固定 $x_i$（行），把所有 $y_j$ 的概率相加
- $Y$ 的边缘分布：固定 $y_j$（列），把所有 $x_i$ 的概率相加
