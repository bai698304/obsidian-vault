# 分布函数

## 定义

设 $X$ 是随机变量，$x$ 是任意实数，称函数

$$
F(x) = P\{X \leqslant x\} \quad (x \in \mathbb{R})
$$

为随机变量 $X$ 的**分布函数**，或称 $X$ 服从 $F(x)$ 分布，记为 $X \sim F(x)$。

## 性质（充要条件）

1. **单调不减性**：$F(x)$ 是 $x$ 的单调不减函数，即对任意实数 $x_1 < x_2$，有 $F(x_1) \leqslant F(x_2)$

2. **右连续性**：$F(x)$ 是 $x$ 的右连续函数，即对任意 $x_0 \in \mathbf{R}$，有

   $$
   \lim_{x \to x_0^{+}} F(x) = F(x_0 + 0) = F(x_0)
   $$

3. **极限性质**：

   $$
   F(-\infty) = \lim_{x \to -\infty} F(x) = 0,\quad F(+\infty) = \lim_{x \to +\infty} F(x) = 1
   $$

   由此知 $0 \leqslant F(x) \leqslant 1$。

> 满足以上三条性质的函数 $F(x)$ 必是某个随机变量 $X$ 的分布函数，这三条性质也是判断函数 $F(x)$ 是否为分布函数的**充要条件**。

## 应用——求概率

| 事件 | 用 $F$ 表示 |
|------|-----------|
| $P\{X \leqslant a\}$ | $F(a)$ |
| $P\{X < a\}$ | $F(a-0)$ |
| $P\{X = a\}$ | $F(a) - F(a-0)$ |
| $P\{a < X < b\}$ | $F(b-0) - F(a)$ |
| $P\{a \leqslant X < b\}$ | $F(b-0) - F(a-0)$ |
| $P\{a < X \leqslant b\}$ | $F(b) - F(a)$ |
| $P\{a \leqslant X \leqslant b\}$ | $F(b) - F(a-0)$ |
