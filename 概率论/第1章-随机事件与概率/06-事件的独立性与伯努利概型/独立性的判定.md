# 独立性的判定

## 基本判据

### 两个事件

$A$ 与 $B$ 相互独立 $\Leftrightarrow P(AB) = P(A)P(B)$

### 等价形式

$A$ 与 $B$ 相互独立 $\Longleftrightarrow$ $A$ 与 $\overline{B}$ 相互独立 $\Longleftrightarrow$ $\overline{A}$ 与 $B$ 相互独立 $\Longleftrightarrow$ $\overline{A}$ 与 $\overline{B}$ 相互独立

## 重要判定规则

1. **替换对立事件**：将相互独立的事件组中的任意几个事件换成各自的对立事件，所得的新事件组仍相互独立

2. **不含相同事件的运算**：对独立事件组不含相同事件作运算，得到的新事件组仍独立。如 $A, B, C, D$ 相互独立，则 $AB$ 与 $CD$ 相互独立

3. **零概率/必然事件**：若 $P(A) = 0$ 或 $P(A) = 1$，则 $A$ 与任意事件 $B$ 相互独立

## 证明要点

$A, B$ 独立 $\Rightarrow P(A\overline{B}) = P(A) - P(AB) = P(A) - P(A)P(B) = P(A)[1 - P(B)] = P(A)P(\overline{B})$
