# 常用分布的期望与方差

| 分布 | 分布列 $p_k$ 或概率密度 $f(x)$ | 期望 $EX$ | 方差 $DX$ |
|------|------|------|------|
| [[0-1分布]] | $P\{X=k\} = p^k(1-p)^{1-k}, k=0,1$ | $p$ | $p(1-p)$ |
| [[二项分布]] $B(n,p)$ | $P\{X=k\} = \mathrm{C}_n^k p^k(1-p)^{n-k}$ | $np$ | $np(1-p)$ |
| [[泊松分布]] $P(\lambda)$ | $P\{X=k\} = \frac{\lambda^k}{k!}\mathrm{e}^{-\lambda}$ | $\lambda$ | $\lambda$ |
| [[几何分布]] $G(p)$ | $P\{X=k\} = (1-p)^{k-1}p$ | $\frac{1}{p}$ | $\frac{1-p}{p^2}$ |
| [[均匀分布]] $U(a,b)$ | $f(x) = \frac{1}{b-a}, a < x < b$ | $\frac{a+b}{2}$ | $\frac{(b-a)^2}{12}$ |
| [[指数分布]] $E(\lambda)$ | $f(x) = \lambda\mathrm{e}^{-\lambda x}, x>0$ | $\frac{1}{\lambda}$ | $\frac{1}{\lambda^2}$ |
| [[正态分布]] $N(\mu,\sigma^2)$ | $f(x)=\frac{1}{\sqrt{2\pi}\sigma}\mathrm{e}^{-\frac{(x-\mu)^2}{2\sigma^2}}$ | $\mu$ | $\sigma^2$ |
