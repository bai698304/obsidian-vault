不同学科的结构化学习笔记仓库。

以教材为骨架、优质 B 站 UP 主内容为补充，按概念而非按来源组织笔记。每个概念遵循 **C→A→B 三层结构**：体系定位 → 内涵与外延 → 实战中的反复出现模式。

## 笔记结构

每个概念一个 `.md` 文件，包含：

```markdown
## 体系定位
前置概念 → 后继概念 → 所属分支

## 内涵与外延
一句话定义 → 核心概念展开 → 经典案例 → 常见误解

## 实战中的反复出现模式
不同来源/不同场景下的同一概念
```
