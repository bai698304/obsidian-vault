## 学科定位

```
GDP（度量起点）
  ├── 长期增长：生产率 → 储蓄/投资 → 货币体系 → 通胀
  ├── 开放经济：贸易/汇率 → 资本流动 → 金融危机
  └── 短期波动：AD-AS → 财政/货币政策 → 通胀-失业取舍
```
