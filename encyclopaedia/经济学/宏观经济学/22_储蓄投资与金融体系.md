# 储蓄、投资与金融体系

> **国民储蓄是经济中用于投资的资源来源。可贷资金市场通过利率协调储蓄者与借款者。** —— 曼昆《经济学原理》第 26 章

## 内涵与外延

### 核心恒等式

**S（国民储蓄）= I（国内投资）+ NCO（资本净流出）**

封闭经济中 NCO=0 → **储蓄 = 投资。** 开放经济中储蓄可以投资国内或国外。

### 可贷资金市场

所有储蓄者和借款者聚在一起的市场。**利率是储蓄的收益和借款的成本。**
- 供给方：国民储蓄（家庭 + 政府）
- 需求方：国内投资（企业建工厂、买房、买设备）
- 均衡利率：储蓄量 = 投资量

### 储蓄的三种形态

| 储蓄类型 | 来源 |
|---------|------|
| 私人储蓄 | 家庭税后收入 - 消费 |
| 公共储蓄 | 政府税收 - 政府支出 |
| 国民储蓄 | 私人储蓄 + 公共储蓄 |

> 政府预算赤字 → 公共储蓄为负 → 国民储蓄减少 → 利率上升 → 私人投资被挤占。这就是"挤出效应"（crowding out）。

### 金融体系的中介作用

果冻AP P24-26：金融体系把储蓄者的资金引导到投资者手中。两种渠道：
- **金融市场**（直接融资）：股票、债券——储蓄者直接购买企业发行的证券
- **金融中介**（间接融资）：银行——储蓄者存款，银行放贷给企业

### 常见误解

- ❌「储蓄多=经济不好（消费不足）」→ ✓ 储蓄是未来的投资和消费的物质基础。一个零储蓄的经济体无法建设工厂、公路、学校。
- ❌「预算赤字没关系」→ ✓ 赤字→政府借贷→推高利率→挤出私人投资→降低长期增长。

## 实战中的反复出现模式

### 模式 1：日本失落的三十年——储蓄过剩的另一面

小Lin说："日本经济——失落的三十年。" 日本拥有全球最高的储蓄率之一→大量储蓄→但国内缺乏投资机会（经济停滞、人口下降）→利率降到接近零→货币政策失效→陷入"流动性陷阱"。高储蓄率通常是好事——但如果缺乏投资机会和大胆的政策，高储蓄可以变成萧条。

### 模式 2：政府赤字如何"排挤"私人投资

当政府借钱（发行国债）为财政赤字筹资时→可贷资金需求增加→利率上升→企业借钱建工厂更贵→私人投资减少。曼昆："预算赤字减少了人力资本（学生的教育）和物质资本（企业的工厂）的投资。由于现在的低投资意味着未来的低生产率，因此，一般认为预算赤字抑制了生活水平的增长。"
