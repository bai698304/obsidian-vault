# 资本流动与金融危机

> **开放经济中，储蓄 = 国内投资 + 资本净流出。当资本流动突然逆转时，金融危机爆发。** —— 曼昆《经济学原理》第 32 章

## 内涵与外延

### 开放经济的核心恒等式

**S（储蓄）= I（国内投资）+ NCO（资本净流出）**

曼昆："每当一国储蓄了一美元收入，它就可以用这一美元来为购买国内资产筹资，或为购买国外资产筹资。" 可贷资金市场的供给来自国民储蓄，需求来自国内投资和资本净流出。

### 资本净流出取决于什么

- **真实利率**：国内真实利率↑ → 国内资产更有吸引力 → NCO↓
- **国外利率**：国外利率↑ → 资本外流 → NCO↑
- **汇率预期**：预期本币贬值 → 资本外逃 → NCO↑
- **政治风险**：政局不稳 → 资本外逃

### 汇率市场：NCO = NX

资本净流出 = 贸易盈余（NX）。买外国资产需要外币→在外汇市场上供给本币、需求外币。利率和汇率在可贷资金市场和外汇市场之间联动——**高利率吸引资本流入，但资本流入推高本币，减少净出口。** 这就是开放经济的"均衡三联锁"。

### 金融危机的几种典型形态

| 类型 | 机理 | 小Lin说案例 |
|------|------|-----------|
| **货币危机** | 固定汇率被投机攻击→外汇储备耗尽→汇率崩溃 | 英镑退出ERM、亚洲金融危机 |
| **债务危机** | 政府/企业过度借外债→无力偿还→违约→银行倒闭 | 希腊、欧债危机 |
| **银行危机** | 资产泡沫破裂→银行坏账→信贷冻结→实体经济衰退 | 2008 全球金融危机 |
| **资本外逃** | 投资者信心崩溃→大规模资本撤出→货币贬值+资产暴跌 | 土耳其、委内瑞拉 |

### 为什么危机会传染

小Lin说覆盖了"资本是如何接连引爆多个国家的金融危机的"——一个国家的违约或贬值→投资者恐慌→重新评估类似国家的风险→资本同时撤出→"自我实现的预言"在多国同步上演。1997 年亚洲金融危机从泰国蔓延到韩国、印尼、马来西亚——每个国家的具体问题不同，但传染机制相同。

### 常见误解

- ❌「金融危机=经济不好」→ ✓ 危机是资本流动突然逆转的结果。"经济不好"可以是慢性的；"危机"是急性的——就像心脏骤停和慢性心脏病的区别。
- ❌「资本流入总是好事」→ ✓ 短期热钱流入→吹大资产泡沫→撤出时泡沫破裂→危机。资本流入的质量比数量重要。

## 实战中的反复出现模式

### 模式 1：希腊——一个国家的"债务死循环"

小Lin说："希腊借了太多的钱→市场怀疑它还不起→利率飙升→借新债成本太高→更还不起→违约。"希腊债务危机本质上是可贷资金市场在开放经济中的极端版本——当储蓄不足以覆盖投资和过去的债务时，只有三个选择：违约、接受救助（丧失财政主权）、或退出欧元区恢复自主货币。

### 模式 2：亚洲金融危机——热钱的"入场-离场"双面杀伤

1997 年：泰国维持固定汇率→大量外资流入→资产价格膨胀→投机者做空泰铢→央行外汇储备耗尽→放弃固定汇率→泰铢暴跌 50%→外资恐慌撤离→银行破产→实体经济崩盘。小Lin说："资本是如何接连引爆多个国家的金融危机的"——这条视频完整复盘了传染链条。

### 模式 3：英国崩盘——政治选择引爆金融定时炸弹

小Lin说："英国崩盘背后的真相"——特拉斯减税方案→市场不信任（财政纪律崩溃）→英镑暴跌、金边债暴跌→养老基金面临追加保证金→几乎崩盘→央行紧急干预。这是财政政策失误如何在开放经济中通过资本流动变为金融危机的典型案例。
