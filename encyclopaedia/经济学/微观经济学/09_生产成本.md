# 生产成本

> **经济成本 = 显性成本 + 隐性成本。经济学家把放弃的机会也算进成本。** —— 曼昆《经济学原理》第 13 章

## 内涵与外延

### 一句话定义

**会计只看账单——显性成本。经济学家看放弃的机会——显性成本 + 隐性成本 = 经济成本 = 机会成本。**

### 会计利润 vs 经济利润

| | 会计视角 | 经济学视角 |
|---|---------|----------|
| 成本包含 | 只有显性成本（看得见的账单） | 显性 + 隐性（放弃的收益） |
| 利润公式 | TR - 显性成本 | TR -（显性 + 隐性） |
| 利润总是 | 更高（或至少相等） | 更低（或相等） |

**果冻AP 柠檬汁店案例**：Molly 卖柠檬汁，TR=1000，显性成本（桌子+柠檬+糖+杯子+执照）=400 → 会计利润=600。但 Molly 放弃了银行 1000/月的工作，且那 400 本金本可存银行赚 10% 利息（40）。隐性成本=1040。经济利润=600-1040=-440。**会计看了觉得赚钱，经济学家看了觉得在亏。**

### 正常利润（Normal Profit）

当经济利润=0 → 会计利润刚好等于隐性成本 → 你的能力恰好被这个行业充分利用，干别的也赚不到更多。此时赚的是"正常利润"——这不是"没赚钱"，是"没亏待自己"。

> 正常利润本质上是机会成本的一部分——它是你选择干这行而放弃的次优收入的货币化。

### 短期 vs 长期

| | 短期 | 长期 |
|---|------|------|
| 工厂规模 | 固定 | 可变 |
| 固定投入 | 存在（厂房、设备） | 全部变成可变 |
| 进出市场 | 不能自由进出 | 可以自由进出 |
| 成本结构 | 固定成本 + 可变成本 | 全部是可变成本 |

> 区分短期和长期不看日历时间——看"你能多快改变你的工厂规模"。果冻AP：Molly 的执照每月 25 块，如果下个月可以不续→"长期"就是大于一个月。

### 短期成本家族

| 成本 | 符号 | 含义 |
|------|------|------|
| 固定成本 | FC | 不随产量改变（租金、设备折旧） |
| 可变成本 | VC | 随产量改变（原材料、工人工资） |
| 总成本 | TC = FC + VC | — |
| 平均固定成本 | AFC = FC/Q | 产量越大摊得越薄 |
| 平均可变成本 | AVC = VC/Q | U 形曲线 |
| 平均总成本 | ATC = TC/Q | U 形，在 AVC 上方 |
| 边际成本 | MC = ΔTC/ΔQ | 先降后升的钩形，穿过 AVC 和 ATC 的最低点 |

### 常见误解

- ❌「会计利润=经济利润」→ ✓ 会计利润总是 ≥ 经济利润。会计不管机会成本。
- ❌「经济利润为零=没赚钱」→ ✓ 经济利润为零=赚了正常利润=你没有被自己亏待。
- ❌「短期就是三个月/一年」→ ✓ 短期=有固定投入的时期。不同行业"短期"长度不同。

## 实战中的反复出现模式

### 模式 1：为什么经济利润比会计利润"悲观"

果冻AP 用 Molly 的柠檬汁店展示了一个反直觉的事实：**同一个生意，会计说有 600 利润，经济学家说有 440 亏损。** 不是经济学家故意悲观——是经济学家把"你本可以做什么"也算进了成本。这个框架直通企业决策：如果一个项目的经济利润为负→你在这个项目上不如去做别的→应该退出。

### 模式 2：MC 曲线为什么重要

边际成本曲线穿过 AVC 和 ATC 的最低点。这不是巧合——是数学必然。当 MC < ATC → ATC 被拉下来。当 MC > ATC → ATC 被拉上去。MC = ATC 时正好是 ATC 的最低点。这个关系在后面四个市场结构的利润最大化分析中反复使用。

### 模式 3：固定成本在生产决策中不该被考虑

果冻AP 在讲 shutdown 条件时强调：已经交了的固定成本是沉没成本——"就算我选择一点都不生产，这些钱我还是要交的"。所以短期停产决策只看可变成本（AVC），不看固定成本。这是机会成本思维的延伸——沉没成本不是成本。
