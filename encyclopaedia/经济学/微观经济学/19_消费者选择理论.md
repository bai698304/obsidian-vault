# 消费者选择理论

> **消费者在预算约束下选择能最大化其效用的商品组合。无差异曲线代表给消费者带来相同满足程度的所有组合。** —— 曼昆《经济学原理》第 21 章

## 体系定位

- **前置**：[[03_边际思维|边际思维]]（边际效用递减是消费者选择的基础）、[[06_供给与需求|供给与需求]]（需求曲线从消费者最优选择导出）
- **后继**：无（微观消费者理论的终点）
- **所属分支**：微观 / 消费者行为

## 内涵与外延

### 一句话定义

**消费者如何在收入有限的情况下，选择商品组合来最大化自己的总效用。** 果冻AP："经济学研究的问题就是如何分配有限的资源来满足人们无限的欲望。这一章就是把这个问题放到每一个消费者的身上——如何最好的分配自己手里的钱，来使自己最开心。"

### 效用（Utility）

经济学家用"效用"来量化满足感——单位是 util。果冻AP："虽然他们也知道这非常主观，开心程度很难量化，但我们可爱的经济学家们依旧不屈不挠，给了开心程度一个量化的指标。"

**总效用（TU）**：从特定数量消费中获取的所有满足感。
**边际效用（MU）**：多消费一单位，总效用的改变量。MU = ΔTU / ΔQ。

关键规律——**边际效用递减**：汉堡第 1 个→MU=10，第 2 个→MU=8，第 3 个→MU=6……第 7 个→MU 变负。"不是汉堡味道变了，是你对汉堡的需求被前面的汉堡满足了。"

### 预算约束

消费者不能想买多少买多少——收入有限。预算线表示在给定收入和价格下能买到的所有商品组合。线内→可达到但钱没花光；线外→买不起。

### 无差异曲线

同一条无差异曲线上的所有点→给消费者**相同的总效用**。消费者在两条无差异曲线之间是无差异的——但曲线越往外（离原点越远）→效用越高。

**四个性质**：
1. 越高越好——外移=效用增加
2. 向右下方倾斜——要更多 X 就得放弃一些 Y
3. 不相交——同一条曲线效用相同，相交意味着矛盾
4. 凸向原点——边际替代率递减（越缺的东西你越愿意用更多另一种东西来换它）

### 消费者均衡

**最优消费组合 = 预算线与最高可达无差异曲线的切点。** 在切点上：MUx/Px = MUy/Py——"花在每一种商品上的最后一美元的边际效用相等"（等边际原理）。

### 收入效应与替代效应

价格变化通过两个渠道影响需求量：
- **替代效应**：商品变贵→你转向替代品→该商品需求量下降
- **收入效应**：商品变贵→你的实际购买力下降→对正常品需求下降，对劣质品需求上升

两者加总 = 需求曲线向右下方倾斜的原因。吉芬商品（Giffen good）是收入效应 > 替代效应的极端特例——涨价反而买更多（极贫困人群中可能存在，如土豆）。

### 常见误解

- ❌「效用可以跨人比较」→ ✓ 效用是序数（可以排序），不是基数（不能说我比你多 5 个 util）。
- ❌「理性=永远买最便宜的」→ ✓ 理性 = 在约束下最大化效用。有人愿意为品牌溢价付更多——那是因为品牌给他带来了额外的效用。
- ❌「无差异曲线一定是直线」→ ✓ 通常是凸的。直线意味着两种商品是完全替代品（如百事和可口可乐对某些人）。

## 实战中的反复出现模式

### 模式 1：消费者选择理论是需求曲线背后的微观基础

果冻AP P15 的讲解逻辑：效用→边际效用递减→消费者在预算约束下最大化总效用→导出需求曲线。**"当你学完这一章，你就知道需求曲线为什么向右下方倾斜——不是因为价格低了大家'自然'买更多，而是因为消费者在最大化效用。"** 这条逻辑链把微观经济学前半部分全部串联起来。

### 模式 2：为什么钻石比水贵——边际效用 vs 总效用

水的总效用极高（没有水会死），但边际效用极低（你已经有足够的水了→再多一杯水→MU ≈ 0）。钻石的总效用不高，但边际效用极高（你几乎没有钻石→第一颗钻石→MU 极高）。价格由**边际效用**决定，不由总效用决定。这是效用理论最经典的反直觉结论。

### 模式 3：替代效应和收入效应在现实中的分离

汽油涨价→替代效应（少开车、多坐公交）+ 收入效应（实际购买力下降）。哪个效应更强取决于商品的性质。对于汽油这种必需品→替代效应弱（短期难以替代）→总收入效应也弱（汽油在预算中占比不大）→总需求量变化不大→需求缺乏弹性。这就是消费者选择理论如何通向[[07_弹性|弹性]]。
