# 完全竞争

> **完全竞争市场：有许多买者和卖者，产品完全相同，每个卖者是价格接受者——任何人都无法单独影响市场价格。** —— 曼昆《经济学原理》第 14 章

## 内涵与外延

### 完全竞争市场的四个特征

1. **许多卖者和买者**——每个人占市场极小份额，单独行动不影响价格
2. **产品完全同质**——张春红和李狗蛋家的鸡蛋没有区别
3. **自由进出**——没有进入壁垒
4. **完全信息**——买者知道所有卖者的价格

→ 每个 firm 是 **price taker**（价格接受者）：市场价格不由你定，你只能接受。面对的需求线是**水平线**（完全弹性）。

### 利润最大化规则

与所有市场结构一样：**MC = MR**。但完全竞争的特殊之处：**MR = P**（每多卖一个产品，收到的就是市场价格，不影响价格）。所以规则简化为：**P = MC**。

### 短期决策：四个区间

| 价格水平 | 决策 |
|---------|------|
| P > ATC | 有经济利润，继续生产 |
| AVC < P < ATC | 有亏损但继续生产（亏的只是固定成本，停产亏得更多） |
| P = min AVC | 停产与继续无差异（shutdown point） |
| P < min AVC | 停产（连可变成本都收不回来） |

> 固定成本是沉没成本——停产决策不看它。只要 P > AVC，每卖一个产品至少可以回收一部分固定成本。

### 长期均衡

利润吸引新企业进入 → 供给右移 → 价格下降 → 利润消失。亏损驱赶企业退出 → 供给左移 → 价格上升 → 亏损消失。最终：**P = min ATC，经济利润 = 0，每个企业赚正常利润。**

### 效率

完全竞争是**唯一**同时达到以下两个效率的市场结构：
- **生产效率**：在 ATC 最低点生产
- **配置效率**：P = MC（价格 = 边际成本 = 社会最优）

### 常见误解

- ❌「完全竞争=有很多竞争对手的商业街」→ ✓ 需要产品同质。服装店虽多但每家产品不同→是垄断竞争，不是完全竞争。
- ❌「亏损就停产」→ ✓ 短期只要 P > AVC 就该继续生产——停产亏的是全部固定成本，生产至少能回收一部分。

## 实战中的反复出现模式

### 模式 1：农民是 price taker 的典型

果冻AP 的案例：张春红把鸡蛋定价 100 块→对市场鸡蛋价格零影响。每个农民只占鸡蛋市场的极小份额，你定价高了没人买，定价低了亏自己。唯一的选择是按市场价格卖、在成本端做文章。

### 模式 2：利润在长期必然消失——即使是暴利行业

果冻AP 的逻辑链：某个行业有经济利润→新企业进入→供给增加→价格下降→利润缩小→直到经济利润=0→进入停止。**"暴利"在完全竞争市场中是暂时的——它不是被政府管制消除的，是被竞争本身消除的。**

### 模式 3：四个市场结构的定位

果冻AP 用竞争激烈程度排列四个市场：完全竞争（最激烈）→ 垄断竞争 → 寡头 → 垄断（最不激烈）。从左到右，firm 对市场的掌控能力（market power）越来越强。完全竞争和垄断是两个极端——学完这两个，中间的垄断竞争和寡头就很好理解了。
