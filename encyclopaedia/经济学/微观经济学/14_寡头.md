# 寡头

> **寡头：只有少数几个卖者的市场，每个卖者的决策会影响其他卖者——企业之间互相依赖。** —— 曼昆《经济学原理》第 17 章

## 内涵与外延

### 寡头市场的特征

1. **少数企业**——"用手指数得过来"：手机（苹果/华为/三星）、石油（中石油/中石化）、运营商（电信/联通/移动）
2. **相互依赖**（mutual interdependence）——每个企业的决策都会引起对手反应
3. **产品可能同质也可能差异化**——石油标准化，手机差异化
4. **进入壁垒高**——需要牌照、资本、关系网

> 果冻AP："四个市场里，寡头的决策逻辑最特殊——你不能一意孤行，要看对手的反应。"

### 寡头 vs 其他市场结构

| | 完全竞争 | 垄断竞争 | 寡头 | 垄断 |
|---|---------|---------|------|------|
| 企业数 | 无数 | 很多 | 少数 | 1 |
| 互相依赖 | 无 | 无 | **有** | 无（就我一个） |
| 策略思考 | 不需要 | 不需要 | **必须** | 不需要 |
| 长期利润 | 零 | 零 | 可正 | 可正 |

### 合谋与卡特尔

当寡头企业合作制定价格和产量 → 合谋（collusion）。正式的合作协议 → 卡特尔（cartel）。合谋后寡头相当于联合成垄断者——共同利润最大化。

**但合谋极不稳定**：每个企业都有**背叛的动机**——别人遵守协议定高价时，我偷偷降价就能抢到更多市场份额。果冻AP："虽然合作后利润最大化，但双方都有背叛协议的动机——这种协议是很难保持的。"

### 常见误解

- ❌「寡头=只有两个企业」→ ✓ 双头（duopoly）是两企业。寡头可以是 3-5 个甚至更多。
- ❌「寡头一定合谋」→ ✓ 合谋是寡头的可能性，不是必然。很多寡头激烈竞争（如手机市场）。
- ❌「寡头一定违法」→ ✓ 只有合谋/卡特尔在很多国家违法。寡头结构本身不违法。

## 实战中的反复出现模式

### 模式 1：你身边全是寡头——手机、石油、运营商

果冻AP P28 列举日常生活里的寡头："手机市场——苹果、华为、三星，用手指头就可以数得过来。石油市场——中国石油和中国石化。运营商——中国电信、中国联通、中国移动。"产品可能差异化（"华为拍照效果好，苹果性能稳定，华为安卓系统，苹果 iOS"），也可能标准化（"石油就是石油，最多加浓度不一样的石油，不会给你加点花生油"）。共通点：**"每个厂商能够影响价格，厂商之间互相影响——你做决策时也要考虑其他公司的决策。"**

### 模式 2：为什么新玩家进不来——进入壁垒

果冻AP："你进入市场会有比较大的障碍——你要有一定的经济实力和关系，才能够得到一张运营许可证。"电信、石油、航空——牌照本身就是壁垒。资本门槛（建一套电信网络要几百亿）和规模经济（新进入者产量不够大、成本压不下去）共同构成了寡头市场的护城河。

### 模式 3：寡头是博弈论最自然的应用场

完全竞争和垄断都不需要策略思维（一个太渺小，一个没对手）。寡头是所有市场结构中唯一需要博弈论的——因为你每走一步都要想"对手会怎么回应？"定价、广告、研发投入都是互动的——这是从寡头通向[[15_博弈论|博弈论]]的直接桥梁。
