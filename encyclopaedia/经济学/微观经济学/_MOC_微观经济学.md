## 学科定位

```
稀缺性（基本假设）
  ├── 个人如何决策：机会成本 → 边际思维 → 激励
  ├── 人们如何交易：比较优势 → 供给与需求 → 弹性
  ├── 政府何时介入：价格管制与税收
  ├── 企业如何生产与定价：生产成本 → 规模经济
  │     └── 四种市场结构：完全竞争 → 垄断 → 垄断竞争 → 寡头
  │           └── 策略思维：博弈论 → 价格歧视
  └── 市场为何失灵：外部性 → 公共品与搭便车
  ├── 消费者如何决策：消费者选择理论（无差异曲线→预算约束→效用最大化）
  └── 要素市场：劳动→资本→土地的定价与收入分配
```
