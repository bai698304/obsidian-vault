# 垄断竞争

> **垄断竞争：许多企业出售相似但不同质的产品，每个企业有某种程度的市场势力，但自由进出使长期经济利润为零。** —— 曼昆《经济学原理》第 16 章

## 内涵与外延

### 垄断竞争的特征

| 特征 | 说明 |
|------|------|
| 较多企业 | 没有完全竞争那么多，但也不少 |
| 差异化产品 | 同类型但不同质——李宁 vs 耐克，都是衣服但商标/设计不同 |
| 有一定定价权 | 因为产品有差异，不是完全的价格接受者 |
| 自由进出 | 与垄断不同，没有壁垒阻止新企业 |
| 广告与品牌 | 用差异化吸引消费者偏好 |

> 果冻AP："垄断竞争同时具有垄断和完全竞争的特征——图画像垄断（MR 在 demand 下方），但长期均衡像完全竞争（经济利润为零）。"

### 需求线的弹性

垄断竞争的需求线处于两个极端之间：不如完全竞争平坦（不是 price taker），但比垄断更有弹性（消费者有更多替代品可选）。

### 长期均衡

短期可能有经济利润 → 利润吸引新企业进入 → 替代品增多 → 每个企业的需求线左移 → 直到 demand 与 ATC **相切** → 经济利润 = 0。

> 与完全竞争的关键区别：长期均衡时 P > MC（配置无效率），且不在 ATC 最低点（生产无效率）——但因为产品差异化，消费者获得了**多样性**作为补偿。

### 常见误解

- ❌「垄断竞争=垄断+竞争混合」→ ✓ 它是一种独立的市场结构。"垄断"指的是产品差异化带来的微小定价权，"竞争"指的是自由进出。
- ❌「做广告就是垄断竞争」→ ✓ 广告是垄断竞争的特征之一，但寡头也做广告。区分看企业数量和进出壁垒。

## 实战中的反复出现模式

### 模式 1：你日常接触的大多数市场是垄断竞争

果冻AP 列举：服装（李宁/耐克）、化妆品、饮料（旺仔牛奶/巧克力奶）、餐馆——都是同类型但彼此不同。这是离真实消费最近的市场结构。

### 模式 2：长期零利润≠和完全竞争一样

垄断竞争长期利润也为零——但与完全竞争本质不同。垄断竞争企业没有在 ATC 最低点生产（有闲置产能），且 P > MC。消费者用**略高的价格**换来了**品种的多样性**。
