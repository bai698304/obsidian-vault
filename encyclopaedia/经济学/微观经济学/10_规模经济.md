# 规模经济

> **长期平均总成本随产量增加而下降时，存在规模经济。长期平均总成本随产量增加而上升时，存在规模不经济。** —— 曼昆《经济学原理》第 13 章

## 内涵与外延

### 一句话定义

**长期中所有投入可变。随着产量扩大，平均成本先降（规模经济）、后平（规模报酬不变）、再升（规模不经济），形成 U 形的长期平均总成本曲线。**

### 短期 vs 长期成本的关系

- 短期：工厂规模固定 → 每条短期 ATC 对应一个工厂规模
- 长期：可以建任何规模的工厂 → 长期 ATC 是所有可能短期 ATC 的**下包络线**
- 每个产量水平上，企业选择平均成本最低的工厂规模

### 规模经济的原因

| 方向 | 原因 |
|------|------|
| 规模经济（下降段） | 专业化分工加深、批量采购折扣、固定成本摊薄 |
| 规模不经济（上升段） | 管理协调成本上升、官僚化、沟通效率下降 |

### 常见误解

- ❌「越大越好」→ ✓ 规模经济有边界。过了最低效率规模，再变大反而不经济。
- ❌「长期成本一定比短期低」→ ✓ 不。长期只是"有更多选择"，但如果你选了非最优规模，长期成本也可以很高。

## 实战中的反复出现模式

### 模式 1：从一家店到连锁——长期 ATC 是怎么拼出来的

果冻AP P18 的推演：一家店赚钱了 → "一家店不够，多开几家分店来扩大规模"→ 每条短期 ATC 对应一个 capital 规模 → 产量到 20 时短期 ATC 回升 → 开第二家店 → 移到更低的粉色 ATC → 产量到 30 → 再扩大 → "每当到了一个新的规模，成本可以继续变低，我就继续扩大"。长期 ATC 是所有短期 ATC 切点连成的包络线——"在每个规模下面，我都会想选择这个规模成本最低的地方"。

### 模式 2：递增→不变→递减——三阶段的必然

果冻AP 的数字：规模经济阶段——"我多投入了 10% 的资源，可以多生产出 20% 的产品"（回报递增）。过了某个点 → diminishing return → 规模不经济——"增加了 10% 的投入，只导致产量增加了 5%"（回报递减）。原因是：**"开了太多分店，规模太大，变得非常难 control 了——和高级管理层交流少，非常不 efficient。"**

### 模式 3：不同行业的长期 ATC 形状不同——它决定了市场结构

果冻AP P18 对比三种 ATC 形状：
- **自然垄断**（如电网）——规模经济区间极长，"基本上都在经历规模经济，到很大的产量才出现不经济"→ 天然只有一家最划算
- **垄断竞争**（如服装）——"规模经济结束以后，会经历很长一段 constant return to scale"→ 大小企业都能活
- **寡头**（如汽车）——规模经济结束快，"规模经济结束之后就会有大幅度的回升"→ 只有少数几家能活

> 这就是为什么农产品市场有成千上万农户（规模经济很小就耗尽），而飞机制造只有波音和空客（规模经济区间极大）。
