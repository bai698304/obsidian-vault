# 垄断

> **垄断：一个企业是其产品的唯一卖者，且该产品没有相近的替代品。** —— 曼昆《经济学原理》第 15 章

## 内涵与外延

### 垄断市场的四个特征

1. **单一卖者**——firm = industry，整个市场只有一个生产者
2. **无近似替代品**——消费者别无选择
3. **价格制定者**（price maker）——垄断者的产量决策会影响市场价格
4. **进入壁垒极高**——法律、专利、资源独占、规模经济等阻止竞争者进入

果冻AP："现实生活中没有纯粹的垄断，就像没有纯粹的完全竞争——它们都是概念模型。"

### 垄断 vs 完全竞争：关键差异

| | 完全竞争 | 垄断 |
|---|---------|------|
| 需求线 | 水平（完全弹性） | 向右下方倾斜 |
| MR vs P | MR = P | **MR < P**（卖更多必须降价，降价影响此前所有单位） |
| 利润最大化 | P = MC | MR = MC（但此时 P > MC） |
| 长期利润 | 零 | **可以持续为正**（进入壁垒阻止竞争） |
| 效率 | 有效率 | **无谓损失**（P > MC，社会最优产量未达到） |

### 为什么 MR < P（垄断者的特殊问题）

完全竞争企业卖多一单位不影响价格 → MR = P。垄断者面对倾斜的需求线 → 想多卖必须降价 → 降价不仅适用于新增的那一单位，此前的所有单位也得跟着降 → MR 低于价格。

> 果冻AP 把垄断称为"四个市场结构中最难的一个"——因为 MR ≠ P，图形分析和直觉相反。

### 自然垄断

当一个行业的规模经济区间覆盖整个市场需求时→一家企业生产全部产品的平均成本低于多家企业分享市场→"自然"形成垄断。例：自来水、电网——建两套管道比一套贵得多。

### 常见误解

- ❌「垄断一定违法」→ ✓ 自然垄断是结构性的，不违法；只有滥用市场势力才违法（反托拉斯）。
- ❌「垄断者想定多高价都行」→ ✓ 价格受需求线约束——定太高没人买。垄断者是价格制定者，不是全能的。

## 实战中的反复出现模式

### 模式 1：垄断者的利润可以永远存在

完全竞争中经济利润被新进入者挤掉——但垄断者有进入壁垒。果冻AP 强调这是垄断与完全竞争最根本的区别：**垄断的利润是"可以长期存在"的**——只要进入壁垒不破。

### 模式 2：现实中的垄断近似——不是纯粹的

果冻AP 给的中国石油、Windows 系统——都不是纯粹垄断，但有垄断特征。Windows 在 PC 操作系统市场曾接近垄断地位——用户切换成本极高，构成事实上的进入壁垒。

### 模式 3：垄断在成本端也可能有效率

自然垄断的情况：与其搞两家水厂、两组管道，不如让一家做然后把价格监管好。垄断的"坏"不在它本身——在它**滥用**市场势力把 P 定得远高于 MC。
