# 公共品与搭便车

> **公共品：既无竞争性也无排他性的物品。搭便车者：享受了公共品的好处却不为此付费的人。** —— 曼昆《经济学原理》第 11 章

## 体系定位

- **前置**：[[16_外部性|外部性]]（公共品是外部性的极端——无人能被排除在收益之外）
- **后继**：无（微观市场失灵终点）
- **所属分支**：微观 / 市场失灵

---

## 内涵与外延

### 物品分类：两个维度

| | 排他性（能阻止未付费者使用） | 非排他性 |
|---|:---:|:---:|
| **竞争性**（我用你就不能用） | 私人品：衣服、巧克力 | 公共资源：清洁空气、街边长凳、渔场 |
| **非竞争性** | 俱乐部产品：软件、付费电视、教育 | **公共品**：国防、街灯、防洪系统 |

### 私人品 vs 公共品

| | 私人品 | 公共品 |
|---|--------|--------|
| 竞争性 | ✅ 我吃了你就没得吃 | ❌ 我用不妨碍你用 |
| 排他性 | ✅ 付钱才能用 | ❌ 无法阻止任何人使用 |
| 谁生产 | 市场（企业） | 必须政府提供 |
| 是否能达到有效数量 | 是（供需均衡） | 否（搭便车问题） |

### 搭便车问题（Free Rider Problem）

公共品一旦提供 → 无法排除任何人使用 → 每个人都有动机"让别人出钱，我白嫖" → **没人愿意出钱 → 市场供给不足 → 必须政府出钱。**

果冻AP："Free rider 就是白嫖——看了不给钱。这导致我们实际上不知道到底有多少人从中得到了好处，供需的均衡点不再是有效的。"

### 公地悲剧（Tragedy of the Commons）

公共资源（有竞争性但无排他性）的问题：渔场谁都能去钓鱼 → 每个人都有动机多钓（"我不钓别人也会钓"）→ 资源耗尽。果冻AP 用草原放牧做例子："每个牧民都不会省着草喂牛羊——久而久之，绿油油的草原变得荒芜。"

解决方案：私有化（把鱼塘卖给私人）、政府管制（发许可证）、社群自治。

### 常见误解

- ❌「公共品=政府生产的产品」→ ✓ 公共品是按"非竞争性+非排他性"定义的，不是按所有权。国防是公共品，国有钢厂不是。
- ❌「教育是公共品」→ ✓ 教育是俱乐部产品——有排他性（要交学费），但非竞争性。不是纯粹公共品。
- ❌「搭便车是不道德的」→ ✓ 个体搭便车是理性行为——问题不在道德，在激励结构。

## 实战中的反复出现模式

### 模式 1：国防是公共品的教科书案例

无论你有没有交税、是不是合法公民，只要你在这个国家的领土内——国防就在保护你。无法排他、不竞争。因此国防不可能由私营企业提供——谁出钱？这就是为什么国防在所有国家都是政府职能。

### 模式 2：灯塔——经济学史上最著名的公共品辩论

萨缪尔森用灯塔论证"必须政府提供"。科斯反驳——19 世纪英国的灯塔其实是私人建造的，通过港口费收费。这个辩论的核心：**现实中很少有"纯粹"公共品——大多数是混合型的。** 关键在于"排他的技术成本"。

### 模式 3：环保是"反公地悲剧"——不排他但竞争

清洁空气没有排他性（谁都能呼吸），但有竞争性（工厂排放废气会减少清洁空气的 availability）。所以需要政府管制——不是"化为私有"（空气没法私有化），而是"限制使用权"（排放许可）。
