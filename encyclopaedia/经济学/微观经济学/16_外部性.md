# 外部性

> **外部性：一个人的行为对旁观者福利的无补偿影响。负外部性使市场产量大于社会最优；正外部性使市场产量小于社会最优。** —— 曼昆《经济学原理》第 10 章

## 内涵与外延

### 一句话定义

**市场交易影响到了买卖双方之外的第三方——而第三方没有参与决定交易是否发生。** 果冻AP 的案例：你买架子鼓，交易在你（消费者）和乐器店（生产者）之间。但你的邻居受到了噪音损害——这就是负外部性。

### 私人成本 vs 社会成本

| | 私人市场 | 社会 |
|---|---------|------|
| 收益 | MPB（私人边际收益） | MSB = MPB + 外部收益 |
| 成本 | MPC（私人边际成本） | MSC = MPC + 外部成本 |

> 负外部性：MSC > MPC → 市场生产**过多**（over-allocation）。正外部性：MSB > MPB → 市场生产**过少**（under-allocation）。

### 无谓损失（Deadweight Loss）

- 负外部性：从社会的角度看，超过 Qsocial 的每一单位产品，MSC > MSB → 社会损失 → 三角 DWL
- 正外部性：从社会的角度看，Qsocial 以内的每一单位产品，MSB > MSC → 该发生的交易没发生 → 三角 DWL

### 三个解决工具（同一个分析框架）

| 工具 | 原理 | 例 |
|------|------|-----|
| **庇古税**（Pigouvian tax） | 征税额度=外部成本 → MSC 内化为 MPC | 碳税、排污税 |
| **科斯定理**（Coase theorem） | 只要产权清晰且交易成本为零，私人谈判可以解决外部性 | 邻居和打鼓者协商赔偿 |
| **排污权交易**（cap-and-trade） | 设定排放总量上限→发许可证→允许交易 | 碳排放交易体系 |

### 果冻AP 的关键教学点：为什么不用配额而用税收

政府可以直接设配额（如限制产量到 Qsocial）。但配额的问题：每个厂商的边际成本曲线不同，硬性平均分配配额 → 低效。税收允许成本低的厂商多生产、成本高的少生产 → 社会以最低总成本达到 Qsocial。（果冻AP P38 详细展开）

### 常见误解

- ❌「外部性一定是坏事」→ ✓ 正外部性也存在（教育、疫苗、美丽的花园）。问题不是"好/坏"——是市场产量偏离社会最优。
- ❌「政府应该控制所有外部性」→ ✓ 科斯定理指出私人也可以解决——如果交易成本够低。
- ❌「外部性=污染」→ ✓ 污染是负外部性的典型例子，但不是唯一。堵车（多一辆车增加所有人的通行时间）、二手烟、甚至好看的花园（正外部性）都是。

## 实战中的反复出现模式

### 模式 1：架子鼓——外部性让你睡不着觉

果冻AP 用架子鼓开讲外部性：你买架子鼓 → 你开心、乐器店赚钱 → 邻居睡不着 → 这笔交易实际上损害了第三方 → 市场在 Qmarket 生产了太多。架子鼓的私人成本只包含了鼓的价格，没有包含邻居的睡眠损失。

### 模式 2：新冠疫苗的正外部性

果冻AP："疫苗只跟你有关吗？不是——它对我们整个社会上的人都能带来好处。你花钱打疫苗，你觉得只提升了你个人的健康，但你也同时减少了社会传播风险。"正外部性 → 市场供给不足 → 需要政府补贴激励更多人打。

### 模式 3：科斯定理的反直觉——谁赔偿谁不重要

打鼓的吵到了邻居。如果邻居有权保持安静 → 打鼓者赔偿邻居的睡眠损失 → 打鼓者会少打鼓。如果打鼓者有权打鼓 → 邻居出钱请打鼓者少打 → 同样结果。**不管权利判给谁，只要交易成本为零，最终产量相同。** 但现实中交易成本从来不为零——这是为什么政府干预有时必要。
