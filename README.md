## 方法论

所有框架文档遵循 **[逐层对齐式批判性框架构建](https://github.com/bai698304/obsidian-vault)**（六阶段法）：

1. 对齐与锚定 → 2. 命题链映射 & 概念审计 → 3. 交叉验证（含最不利证据检索）→ 4. 冷重读审计 → 5. 策略推导 → 6. 输出

每篇文档标注方法论版本（V1–V7），见 `method` frontmatter 字段。
