## 方法论

本仓库中的框架文档遵循 **[逐层对齐式批判性框架构建](https://github.com/bai698304/obsidian-vault)** 方法论（六阶段法）：

1. 对齐与锚定 → 2. 命题链映射 & 概念清晰度审计 → 3. 交叉验证（含最不利证据检索）→ 4. 冷重读审计 → 5. 策略推导 → 6. 输出

每个框架文档标注了方法论版本（V1–V7），详见各自的 `method` frontmatter 字段。
