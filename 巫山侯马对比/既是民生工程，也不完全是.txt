既是民生工程，也不完全是

  ---
  为什么说是民生工程

  巫山新城的地形决定了它必须建。

  旧城被三峡水库淹掉之后，新城建在老城上方的山坡上。巫山县城是一个垂直城市——从长江边的滨江路（低）到山顶的工业路（高），
  直线距离不长，但高差约242米，相当于80层楼。

  原来的解决方式是神女大道——1136级台阶。从下走到上需要约1小时。

  每天走这条路的是什么人：
  - 住在下半城、上班在上半城的人
  - 去县政府办事的居民（县政府在广东中路，半山腰）
  - 去医院看病的人（医院沿线分布）
  - 学生（学校沿线）
  - 去码头/车站的人

  800米范围内直接服务4万人以上日常出行。 对一个45万人口的县城来说，这覆盖了将近十分之一的人口。把1小时台阶缩短为扶梯上的
  20分钟，这是实打实的民生需求。尤其是老年人——巫山65岁以上占17.78%，你让他们爬1136级台阶，不是在要命吗。

  地形逼出来的需求，不是锦上添花。

  ---
  为什么不完全是民生工程

  看三个细节：

  1. 规模超过了纯民生需求

  21部扶梯 + 8部电梯 + 4部自动人行道 + 2座人行天桥，总长905米、提升242米。对比重庆主城的皇冠大扶梯（全长112米、提升52.5
  米）——巫山这个长度是皇冠的8倍，高度是4.6倍。

  如果纯粹为了解决出行，不需要搞成"世界级"规格。巫山2003年的第一版规划就直接对标"世界级大扶梯"，当年的设计长度1380米——比
  最终建成的还长近500米。说明从一开始，这就是一个城市名片工程，民生是基础，但野心不止于民生。

  2. 明确植入了旅游功能

  巫山县城乡规划建设管理委员会2024年第五次会议明确要求："有效植入巫山文化、旅游元素，通过文化、艺术手段，打造集休闲娱乐
  、观光赏景于一体的文化广场，并与滨江的自然风光有机融合，形成一幅动人心魄的景观画卷，成为市民游客打卡的新地标。"

  "市民游客打卡的新地标"——这句话说明它不是纯粹的市政交通设施。它是一个混合体：底层是民生（解决出行），中层是城市更新（老
  旧小区交通提升项目的名头），顶层是旅游（新地标、打卡点）。

  3. 资金来源决定属性

  项目名称为"高唐组团老旧小区交通出行提升工程"，走的是城市更新的资金渠道，也就是债券资金+市级补助。但如果没有三峡后续政
  策的资金池支撑，单靠巫山本级财政（年收入13亿、自给率24%）根本建不起这个规格的项目。

  ---
  所以准确的定性是什么

  ┌──────────┬─────────────────────────────┬──────────────────────┐
  │   层面   │            属性             │         判断         │
  ├──────────┼─────────────────────────────┼──────────────────────┤
  │ 功能性   │ 替代1136级台阶，4万人受益   │ 纯民生               │
  ├──────────┼─────────────────────────────┼──────────────────────┤
  │ 地形约束 │ 新城建在陡坡上，别无选择    │ 历史欠账的补课       │
  ├──────────┼─────────────────────────────┼──────────────────────┤
  │ 规格选择 │ 905米/21部扶梯/8部电梯      │ 远超纯民生需要       │
  ├──────────┼─────────────────────────────┼──────────────────────┤
  │ 设计意图 │ 植入旅游元素，打造打卡地标  │ 城市名片+旅游功能    │
  ├──────────┼─────────────────────────────┼──────────────────────┤
  │ 资金来源 │ 城市更新项目+三峡后续资金池 │ 政策驱动，非本级财力 │
  └──────────┴─────────────────────────────┴──────────────────────┘

  结论：70%民生工程 + 30%城市名片。

  巫山确实需要解决垂直出行问题——这是三峡移民搬迁的后遗症，1136级台阶不是人可以日常承受的。如果只建一套普通的升降系统（比
  如几部直梯+空中连廊），那就是100%的民生工程。

  但巫山选择了最大规格来做这件事——把民生的"需要"做成了城市的"地标"。这背后的逻辑也不难理解：巫山是旅游名县，任何大型公共
  设施都要考虑旅游价值和城市形象。就像重庆把轻轨穿楼做成了景点一样，巫山想把大扶梯做成景点。

  说到底，如果巫山老城没被淹，根本不需要建这个东西。三峡工程逼出了一座垂直城市，垂直城市逼出了世界上最大的扶梯系统。这个
  扶梯，既是民生，也是三峡移民历史的纪念碑。
