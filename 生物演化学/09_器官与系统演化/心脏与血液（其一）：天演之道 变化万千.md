## 内容概要

心脏和循环系统从假体腔中的变形细胞（爬来爬去搬运营养）起步，到真体腔肌肉挤压假体腔形成血管、血管蠕动集成为心脏。节肢动物和软体动物走向开管式循环让血液浸润器官，节肢动物更进化出气管系统彻底舍弃血液运氧——这一革命让昆虫成为陆地上代谢效率最高的动物类群。

---

## 演化脉络

### 一、最原始的"循环"：变形细胞搬运工

- 6亿年前，原始动物（类似丝盘虫）：
  - 细胞可自由变换形态
  - 遇到食物时形成临时消化腔
  - 后代将消化腔常备化→**内胚层**（肠道）与**外胚层**（表皮）
  - 内外胚层之间的空隙→**假体腔**

#### 变形细胞
- 未固化的细胞，承担搬运工角色
- 从内胚层细胞吞入营养颗粒→爬行到表皮附近转交
- **最原始的循环系统**就是这些爬来爬去的变形细胞
- 今天水母等动物仍在使用这种运输机制

### 二、静态液压骨骼与真体腔

- 动物体型增大（突破厘米级）→需要对抗重力、水流等
- **肌肉 + 骨骼**：最初肠道表面的肌纤维→包裹肠道→向表皮扩展
- 肌肉包裹的中空管腔→**真体腔**
- 高压流体使动物身体如充气轮胎般坚韧→**静态液压骨骼**（最早的骨骼模式）

### 三、血管与心脏的诞生

| 结构演变 | 说明 |
|---------|------|
| 真体腔扩张 | 挤占假体腔空间 |
| 假体腔 | 被压缩为真体腔中的狭窄缝隙/细小的管道→**最早血管** |
| 变形细胞 | 被限制在血管中→专职搬运与免疫→**最早血细胞** |
| 肌肉包裹血管 | 牵引血管蠕动→**血液循环** |
| 血管蠕动集成 | 粗大血管周围肌肉集中→**心脏** |

- **腐仙狐虫**（5.16亿年前）已发现完备的心脏与循环系统
- 两侧对称动物的共同祖先可能已有心脏

### 四、血氧蛋白的出现

- 血氧蛋白（血红蛋白、血蓝蛋白等）比血液循环更古老——细菌中也有
- 早期动物体积小→通过表皮直接扩散氧气即可
- 随着体型增大+生存策略分化→需要血氧蛋白遍布全身血管

#### 血红蛋白的困境

| 蛋白质 | 吸氧能力 | 问题 |
|--------|---------|------|
| 血红蛋白 | **最强** | 分子巨大→直接溶解使血液粘稠；存在于血细胞中又阻碍变形能力 |
| 血蓝蛋白 | 较弱 | 副作用小，早期动物首选 |

- 马尔拉虫（5.05亿年前）找到最早血液痕迹——和我们螃蟹一样流淌淡蓝色血液（血蓝蛋白）

### 五、开管式循环：节肢动物与软体动物的解法

#### 闭管式局限
- 血液只在封闭血管中流动→物质/氧气须穿透两层细胞才能到达目标
- 早期心脏羸弱→血流缓慢

#### 开管式循环
- 打通血管与真体腔之间的壁垒
- 血液在血管末端直接流进真体腔→充分浸润内脏和肌肉→**扁平化传输**
- 传输完物质后→在身体另一侧重新吸回血管→送至鳃与消化道
- 强健心脏通过高压与负压驱动，瓣膜确保单向流动

### 六、节肢动物：开管式的巅峰与转向

| 阶段 | 特征 |
|------|------|
| 巅峰 | 毛细血管以细胞级精度铺展到全身每一个角落 |
| 优势 | 保证肌肉/神经/内脏均匀浸润→最迅捷敏锐的生物类群 |
| 登陆困境 | 外鳃不能呼吸空气，皮肤呼吸浪费水分，血蓝蛋白运氧低下 |

#### 革命性解法：气管系统

> **气物分流**：完全舍弃血液循环系统的运氧功能，另建独立气管网络

- 身上开若干气孔→通过腹部胀缩呼吸空气
- 气孔之内庞杂气管→分叉再分叉→毛细气管直接送氧到每个细胞
- 缔造**自然界最强悍的氧气输送能力**
- 陆地节肢动物（昆虫）代谢效率达到水中同类望尘莫及的高度
- 翅膀意外演化→一飞冲天，至今仍是种类最繁盛的动物类群

## 关键概念

| 概念 | 说明 |
|------|------|
| 变形细胞 | 最原始循环系统——爬行搬运营养颗粒的细胞 |
| 假体腔→真体腔 | 内外胚层之间的空间（假体腔）被肌肉包裹的真体腔挤压→形成血管 |
| 静态液压骨骼 | 高压流体使动物身体坚韧如充气轮胎，最早的骨骼模式 |
| 开管式循环 | 血管末端开放到体腔，血液直接浸润器官（节肢/软体动物） |
| 闭管式循环 | 血液始终封闭在血管中（脊椎动物） |
| 气管系统 | 节肢动物废弃血液运氧，另建独立气管网络直接送氧到细胞 |
| 血红蛋白 | 吸氧能力最强但分子巨大、使血液粘稠的蛋白质 |
| 血蓝蛋白 | 吸氧能力较弱但副作用小的蓝色血液蛋白质 |

|------|-------------|---------|
| 水母等刺胞动物 | 变形细胞搬运 | 最原始"循环"机制 |

对嗯，朋友一定很想知道历史上第一颗心脏是怎么开始跳动的，对吧？这是六亿年前地球当此之时，生物圈静默无声。甚至于连我们后来所知的那些长相肉饼肉串一样的所谓艾迪卡拉动物都尚未出现苍茫的远古大海之下，只有无数微生物铺就而成的后世菌毯，以及偶尔生长于其上的叶状藻类。然而就是在这样的海底，却有一些宛如来自混沌异识空的诡异生物。它们的形态有点类似于今天的丝盘虫，构成它们的每一个细胞都像史莱姆一样，可以自由变换形态，继而带动，其整体也没有固定的形状。当他们遇到食物的时候，就会形成一个临时的消化腔，将食物包裹其中消化吸收。不管你信不信，这就是我们今天绝大多数动物的祖先久而久之，这些变形生物有一部分后代决定，直接将这个消化腔常备化。从此，组成这些生物的大部分细胞不再能自由变化，而是固定成两部分，大约一半细胞永久性的卷在内部演化，成了我们今天意义上的肠道或者专业点叫做内胚层。而另一部分细胞则固定在了外部演化成原始的表皮，也可以叫做外胚层。而内外胚层之间的空隙则被称为假体腔。就是在这么一个假体腔中，心脏的种子却已经萌芽。因为有一些细胞并没有像内外胚层中的细胞那样固化，而是执拗的继续着和祖先一样能够自由变换形态，这些细胞才是这个原始动物真正的灵魂所在。因为他们要承担除了肠道和表皮细胞以外的几乎一切职责，他们得不断维持旺盛的分裂。在繁殖时期，他们还要分化产生临时性的生殖器来让这个动物繁殖。而一旦有病原体入侵这个动物体内，这些变形细胞又会转变成战士去吞噬杀死外来的病原体，今天存在于我们体内的巨噬细胞、中性粒细胞等等，还依旧保持这些原组的变形能力，忠实的履行着吞噬病原体。这个绵延六亿年的古老职责，但这些变形细胞最日常的职责却是搬运工。他们从肠道内胚层细胞那里吞入打包好的营养颗粒，然后自己爬到表皮那边，将这些营养颗粒再转交给外胚存细胞。如果一定要追根溯源的话，最原始的循环系统就是这么几个爬来爬去的变形细胞，直到今天如水母这些动物还在使用这样的运输机制。但器官演化时，在告诉我们，一切生物演化都是将原有的结构不断，小修小补无数的一小步积累起来，就是开天辟地的一大步。尽管由于几乎没有任何化石证据导致循环系统演化一直迷雾重重，但还是有人根据这个规律，提出了一个非常有趣的假说，随着生存竞争日益严酷，当时几乎所有的动物都开始急剧变大而变大，就必须要突破一个关键障碍，维持形态低于毫米级的早期动物仅仅依靠细胞自身的强度，便足以维持自身形态。但是当动物的体型突破厘米级的时候，巨大体型所遭受的重力、浮力、水流冲击力、沙石摩擦力等等，就足以扭曲整个生物体的结构。为了解决这个问题，几乎所有动物都给出了唯一的最优解肌肉与骨骼最原始的肌肉，不过是附着在肠道表面，几根纤维，通过有节掠收缩来让肠道蠕动，而现在肌肉组织变得日益发达。他们先是大致包裹住了整个肠道，随后就向表皮扩展，从而在原本的假体墙势中构建出了一根根中空的肌肉管子，这些完全被肌肉包裹的管腔被称为真体腔。我们知道如果给一个密闭空间里的流体，一个很大的压力，比如打足了漆的轮胎，就会让这个物体变得非常坚韧。早期的动物就利用这个原理，相比于只有薄薄两层细胞包裹的假体腔包裹，真体腔肌肉要结实的多，足以承载高压液体，让动物的身体像打足了卸轮胎一样，维持自身形态，这就是动物最早的骨骼模式被称为静态液压骨骼，在其他骨骼形式出现之前，静态液压骨骼往往要充满全身。于是乎，动物的真体腔日益发达，反客为主，挤占掉了假体腔的空间。原本的假体腔反而变成真体腔中的狭窄缝隙，或者用更确切的词汇来描述。这是真体腔中细小的管道。对，这就是最早的血管，而原本存在于假铁枪中的一部分。变形细胞也从此专门负责搬运物质与免疫，吃上最早的血细胞。更重要的是，这时候从结构逻辑上讲，血管也被肌肉包裹着。于是就有一部分肌肉独立了出来，就像当初牵引肠道蠕动一样，开始牵引血管蠕动，从而带动其中原始的血液在血管中单向循环流动，从此便有了血液循环，血细胞终于不用再亲自跑路了。最后，最后驱动血管蠕动的肌肉逐渐集成到了某几根粗大血管周围，这样既能节省能量肌肉之间也会更容易协调，而这终于缔造出了循环系统的核心心脏。这个演化过程是如此顺滑，几乎是在地质时间尺度的一瞬间就完成了。在生活于距今五点一六亿年前的腐仙狐虫身上，我们就已经发现了完备的心脏与循环系统。而且从发育和分子生物学证据来看，我们两侧对称度的共同祖先体内，说不定就已经有了一颗或几颗澎湃的心脏。现在距离我们所熟悉的血液循环只差最后一块病毒碎片了，氧气供应与我们的直觉不同。我们今天所知道那些在血液中负责运载氧气、血氧蛋白，比如红色的血红蛋白、蓝色血蓝蛋白与玫瑰、红色、蚯蚓、血红蛋白等等，其实都比血液循环本身要古老的多，甚至在某些细菌中都能发现他们的踪迹迹。不过最早的动物并不怎么需要专门去输送氧气，毕竟借由表皮原石塞氧气本来就能均匀扩散到全身。如果直接扩散不够，那就给某些特别需要氧气的器官，再额外配置一些血氧蛋白就可以了。比如说有些线虫就会在一起口部存储很多血红蛋白来保证它在缺氧环境中也能正常进食。所以直到今天，诸如文昌鱼这些动物还依旧保持着那种最圆形的循环系统，并不怎么依靠血液来输送氧气。然而，随着寒武纪海洋的日益拥挤，动物开始发展出了不同的生存策略。有的通过体型增大以物理服人，那他就需要有专门的渠道，把氧气从体表或呼吸器官输送到全身。有的选择固着绿石，他们的大半截身子都埋在缺氧的菌毯之中，同样需要来自触手的氧气供应，而原本固定于某些器官中的血氧蛋白放进遍布全身的血管，就是对此最快也是最优的解法。动物体内吸氧能力最强的蛋白质毋庸置疑是血红蛋白，只不过血红蛋白分子过于巨大，直接溶解。在血液中的话，会让血液变得过于粘稠，难以流动，存储于血细胞中，又会严重阻碍血细胞的变形能力，加之其在寒冷环境中性能不佳。所以在一开始几乎没有任何动物选择将血红蛋白作为血液的氧气载体，而是改用吸氧性能差一些，但没啥副作用的血蓝蛋白或是蚯蚓、血红蛋白等等。由此，我们今天意义上的血液登上历史舞台。在距今五点零五亿年前的马尔拉虫身上，我们就找到了契井已至最早的血液痕迹。它和我们今天的螃蟹雨后一样流淌着淡蓝色的血液。好了，现在有心脏了，有血液了，够了吗？对于绝大多数动物来说确实够了。除非这个动物想要登上霸主职位，而两亿年来不断谋求霸主保座的，基本上只有节肢动物、软体动物和我们脊椎动物，而是育三家也将开创循环系统的三大最强版本原始形态循环系统对于追求更快、更强。动物霸主而言，太过低效，血液只在封闭的血管中流动，就意味着营养物质和氧气，至少需要穿透两层细胞才能抵达目的细胞。而早期羸弱的心脏，也只能让血液很缓慢的流动循环系统需要一轮彻底的革命。而最先发育的软体动物和节肢动物都不约而同的给出了一样的答案。开管式循环，他们直接打通了血管和真体腔之间的心之臂。障血液在血管末端直接流进真体腔，完全浸润，内脏和肌肉，实现了物质与氧气的扁平化传输传输完物质的血液，又会在身体另一侧被重新吸回血管，并快速送到塞与消化道附近，再度流出血管浸润。这些器官高效装填。而为了让血液充分压进组织间隙，与快速吸回血管，他们又演化出了一颗特别强壮的心脏，通过有力的搏动，不断产生高强度的镇压与负压。同时，在各大血管中都配备了瓣膜来确保血液的单向流动，而秉承远古科技一波流磁路的截肢动物，更是将开管式循环进化到了巅峰。他们为自身构建起了极其庞杂的血管网络，以细胞节精度将毛细血管铺展到了身体的每一个角落。在物理上确保了身体的主要肌肉神经和重要内脏都能获得均匀的血液浸润，让节肢动物在相当长的一段时间内，都是地球上最迅捷、最敏锐的生物类群，而节肢动物自带的保水外骨骼，也让他们成了动物界登上陆地的先锋。而在陆地上，他们又遇到了新的矛盾，节肢动物引以为豪的外塞，几乎不能在陆地上呼吸空气。而用皮肤呼吸又会浪费宝贵的水分。同时，截肢动入开管式循环，血流缓慢，其中的血蓝蛋白运载能力又过于低下。这一道道历史珍科，宛如一道道封印，阻止了节肢动物在陆地上的开拓之路。然而，几乎所有的泛截肢动物都用同一招一次性解决了以上所有问题。气物分流简单来说，他们几乎完全舍弃掉了血液循环系统的输送氧气功能，转而让其全心全意负责输送营养与免。嗯，与此同时，他们又另外演化出一套气管系统，他们在身上开了若干气孔，直接通过腹部的胀缩来呼吸空气，而气孔之内则是庞杂的气管。这些气管就像血管一样，分叉再分叉，直至变成无数毛细气管，直接把空气喂到每一个细胞之内，对与其煞费苦心的去改造。原来那套废拉不堪的循环系统，干脆不破不立，彻底革命演化总是能在出人意料的地方，找到最快的结晶。于是，气管系统缔造自然界最强悍的氧气输送能力，让陆地节制动物的代谢效率，直接冲到其水中氢气望尘莫及的高度。在此代谢加持之下，昆虫变化出了强悍的肌肉，驱动着它们意外演化而来的翅膀。代表动物历史上第一次一飞冲天，也在世间一切角落，安家落户至今仍是地球上毋庸置疑，种类绝对最满身的动物类群。然而，在节肢动物一路势如破竹，同时丧尸先界软体动物和脊椎动物又会在循环系统上用什么方式弯道超车，打破节肢动物的垄断呢，这就让我们下期再来讲述吧。
