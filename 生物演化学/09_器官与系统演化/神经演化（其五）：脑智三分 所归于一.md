# 神经演化（其五）：脑智三分 所归于一

## 内容概要

脊椎动物的三段式脑（前脑→嗅觉、中脑→视听觉、后脑→运动）经历了一场漫长的权力斗争。前脑嗅觉中枢（端脑）凭借区位优势逐步侵夺中脑视觉处理权和后脑运动审批权，最终成为主导。龙兽两族分化出截然不同的高级脑架构——兽族的六层新脑皮质（高处理效率）与龙族的五核团背侧室脊（高灵活度），分别走向不同的智力演化路径。

---

## 演化脉络

### 一、始祖设定：三段脑

| 脑区 | 原始功能 |
|------|---------|
| 前脑 | 嗅觉 |
| 中脑 | 视听觉 |
| 后脑 | 运动 |

寒武纪霸主：泛节肢动物（奇虾等）+ 头足类软体动物。脊椎动物祖先只是口粮。

### 二、甲胄鱼时代：嗅觉为王

- **甲胄鱼**（4.7亿年前奥陶纪）：白天埋沙避敌，晚上活动
- 第三只眼（松果体）→只能感知明暗，现在仍存在于脑中
- 黑暗环境 → 视觉受限，听觉未充分发展 → **嗅觉是唯一依靠**

#### 嗅球诞生
- 前脑向前伸到嗅觉感受器附近 → 两个"兔子耳朵"（原始嗅球）
- 七鳃鳗的嗅球已经明显膨胀，松果体仍保持眼结构
- 气味信息成为最高纲领 → **嗅球急剧膨胀为端脑**
- 原前脑被夹在中间 → 成为感觉汇总中心→**间脑**
  - 背侧→**丘脑**（感觉汇总上传）
  - 腹侧→**下丘脑**（内分泌总指挥，主管本能）

### 三、视觉崛起与两强争霸

- 奥陶纪末大灭绝（4.4亿年前）→ 甲胄鱼向浅水上移
- 白天活动→ **视觉重要性提升**
- 中脑视觉细胞急剧扩编→**视顶盖**
- 丘脑的尴尬：不善于处理视觉→被架空为"秘书"
- **端脑 vs 视顶盖**：两强共治

| 阵营 | 优势 |
|------|------|
| 端脑（嗅觉党） | 历史悠久，细胞种类多，底层架构掌控者 |
| 视顶盖（视觉党） | 视觉最重要，炙手可热 |

### 四、后脑逆袭：髓鞘与运动革命

后脑最初只是运动执行端，却因为两件事改变了历史走向：

#### 1. 偶鳍（骨甲鱼）
- 从简单摆尾到花式游泳 → 运动复杂度飙升

#### 2. 髓鞘
- 包裹神经纤维的绝缘层 → 跳跃式传导
- 裸露神经：~1m/s → 髓鞘神经：**最快100m/s**

| 动物类群 | 是否有髓鞘 | 为何只有脊椎动物受益 |
|---------|-----------|-------------------|
| 对虾 | 有，结构更复杂 | 但神经架构为"地方模式"，髓鞘带不动基层 |
| 脊椎动物 | 有 | 先中央后地方→神经纤维从脑直接延伸到脚底→髓鞘使中枢高效传令 |

- 后脑急剧膨胀 → **脑桥**（感觉运动分流中心）
- **小脑**：从无到有构建，含巨量神经元，神经细胞数量**与大脑皮层持平**

### 五、有颌类的出现：端脑独大

- 下颌出现（4.2亿年前）→ 口腔空间扩充
- 端脑近水楼台→**急速膨大**
- 视顶盖被端脑阻挡+小脑挤压→发展空间受限

#### 端脑夺取视觉

| 阶段 | 视觉处理 |
|------|---------|
| 原始 | 直接传入视顶盖 |
| 过渡 | 先经过丘脑→端脑有了"截胡"机会 |
| 最终 | 视觉处理向端脑转移→视顶盖萎缩为上丘（仅处理扫视等杂活） |
| 听觉同理 | 端脑从后脑夺取听觉处理权→中脑下丘退化 |

### 六、端脑内部分化

- **背侧**→皮层（感觉整合）
  - 内侧皮质→海马体+内嗅皮质（记忆）
  - 背侧皮质→思维与决策（智慧核心）
  - 外侧皮质→梨状皮质（嗅觉，最古老的本行）
- **腹侧**→纹状体（运动审批）

#### 纹状体功能

| 神经细胞类型 | 功能 | 病变 |
|-------------|------|------|
| 运动开始指令细胞 | 发起运动 | 帕金森病（功能失常→难以发起运动） |
| 运动终止指令细胞 | 停止运动 | 亨廷顿舞蹈症（功能失常→停不下来） |

- 小脑受损→**共济失调**（运动精度降低，笨拙摇晃）

### 七、龙兽争霸：两种大脑架构

#### 龙族（蜥形纲→鸟类）

| 特征 | 说明 |
|------|------|
| 架构 | 5群神经细胞分别聚集成5个神经核团→**背侧室脊** |
| 方式 | 一切意识活动拆解为任务，分配给各核团 |
| 优势 | 灵活：算力按需分配，小脑也能产生较高智力 |
| 代价 | 核团满负荷时所有任务排队→效率不稳 |
| 扩展 | 背侧皮质向内拱起，填满端脑空腔，再反向往外顶出 |

#### 兽族（合弓纲→哺乳类）

| 特征 | 说明 |
|------|------|
| 架构 | 5群神经细胞按六层规律叠放→**新脑皮质** |
| 方式 | 每个意识活动有专属功能区（如视觉皮层、语言皮层） |
| 优势 | 处理效率极高 |
| 代价 | 新增智力活动需新增皮层→算力利用率低，大脑皱巴巴 |
| 扩展 | 背侧皮质向外凸起翻折，包在端脑外面 |

### 八、盲视与意识

- 大脑损伤导致视觉无法传递到端脑（仅到上丘）
- 患者能躲避障碍、拿取物体、说出眼前东西的名字
- 但声称自己是盲人——**盲视**
- 暗示：意识可能是信息在端脑汇总时产生的特殊感觉

### 九、归于一

两种架构都在现实中产生了大量智力超群的动物。灵长类最终演化出唯一创造过文明的物种——人类，使得亿万年的演化造物能够理解并分享亿万年的演化本身。

## 关键概念

| 概念 | 说明 |
|------|------|
| 松果体 | 甲胄鱼的第三只眼，现代已无视觉但仍是生物节律中枢 |
| 嗅球→端脑 | 原始嗅觉处理中心逐步膨大，成为脊椎动物高级认知中心 |
| 丘脑 | 感觉信息上传中继站，原为嗅觉处理区后被架空的"秘书" |
| 下丘脑 | 内分泌总指挥，掌管本能活动 |
| 视顶盖 | 中脑膨大形成的视觉处理中心，后被端脑夺权萎缩为上丘 |
| 髓鞘 | 绝缘层使神经信号跳跃式传导，速率提升100倍 |
| 纹状体 | 运动审批中心：决定运动开始（帕金森病）与结束（亨廷顿舞蹈症） |
| 小脑 | 精细运动调控，神经元数量=大脑皮层 |
| 新脑皮质 | 兽族六层结构，每项意识活动有专属功能区 |
| 背侧室脊 | 龙族五核团架构，所有意识活动共享算力 |
| 盲视 | 视觉信息进入上丘但未进入端脑意识层面的现象 |

说起来很离奇，我们今天聪明才智，或许源自我们演化史上最窝囊的一段历史书，接上回我们脊椎动物的祖先。经过一番大破大裂，终于演化出一个三段式脑子。这个脑子从前往后，分别是主要负责嗅觉的前脑，主要负责视听觉的中脑和主要负责运动的后脑三部分。这让我们的祖先有了那个时代最优秀的神经架构，可谓开局赢一半，但也就剩了开局。当时地球霸主毋庸置疑，属于以鳍虾海蟹为代表的泛节肢动物鱼，以绞食为代表的头族类软体动物。你可以猜一猜，他们有没有考虑过退位动衔？而男祖先是比较是食物的，当场就入土了，他叫新甲鱼。我们在四点七年前奥淘级的祖先大咋知长，这副棒槌一样，一般被笼统的称为甲咒鱼。他们一般白天会把自己埋进海底沙土里，躲避敌害，晚上才敢出来，过点夜生活。那么问题来了，你就这么修炼土遁忍术，你咋知道现在是白天还是黑夜呢？很简单，许多假胄鱼头顶上都长了一颗小眼睛，专门负责偷瞄。现在天色明暗，这只第三眼，直到今天依然存在于我们的脑子中。虽然它已经不再有任何势力，但依旧是我们生物节律中枢的核心洞见之一，被称为松果体。不过这晚上黑灯瞎火的视觉目测作用有限，而听觉在当时还没充分发展起来，因此唯一能指望的就只有嗅觉了，只不过吧，海洋里各种物质成分复杂，我们的祖先为此得配置多种不同的嗅觉感受器，同时还得根据不同位置海水中气味的细微差异来锁定食物预警敌害，这个数据量就有点大了。这倒不是说我们那个时代祖先脑子不够用，只不过当时神经细胞的传输速率比较慢，带宽有点小捉急。不过这事儿好解决，就像节肢动物，把一部分脑子塞进附眼里，就地计算一样。咱们祖先也把一部分前脑向前伸到了嗅觉感受器附近。而这就让元蛇三段脑最前端衍生出两个人兔耳朵，我们姑称之为原始绣球吧。直到今天，我们脑子也还依然像我们五亿年前祖先一样，把小触角色绣球伸到鼻腔附近。只不过我们今天的绣球和最初的那个原始绣球之间有一点小差别。大家可以猜一下，最初那个原始绣球，对应于我们今天脑部的哪些部件答案是我们的绣球外加整个大脑以及一部分脑干。对，就是这么离谱，这是齐塞满的稿子。虽然他总体上还维持了脊椎动物组件三段脑模式，但可以看出他的绣球已经明显膨胀了。顺便说，七赛满的松果体还保持眼睛的结构，甚至还连着是神经。而我们在四点八亿年前，祖先或许就长了类似这样的脑子。由于长期依赖嗅觉活动，无情中嗅觉信息就成了一切行为的最高纲领。为了精益求精的提升，嗅球精度绣球的规模也随着急剧膨胀，最终形成一个新的完全独立的脑区，被称为端脑。然而，随着嗅觉处理的核心转移到绣球或者说端脑。那么原本负责处理嗅觉的前脑就被闲置了，但没事儿他还狠在端脑与其他脑区之间所有来自端脑的嗅觉信息，必须经过自己才能传达到其他地方。虽然说把神经演化比作权力斗争不太恰当，但不得不说啊，前脑的区位优势实在是过于近水楼台先上菜了。于是，原本的前脑就顺势变成了一个感觉。信息的汇总中心被称为尖脑。由于大部分感觉神经都是从背侧传入肩脑。因此接脑的信息汇总功能也逐渐向被侧集中形成一个叫做丘脑的脑区。这个脑区在我们今天的脑中也还存在位于这里，而剩夏不怎么能直接接受感觉神经投射的复侧部分，肩脑也没闲置，将其改造成了总揽内分泌大臣的激素指挥中心，继而也掌管了脊椎动物的大部分本能活动被称为下丘脑。就我们今天的脑中下丘脑也依旧在继续着他五亿多年来的职能。就这样，一手抓神经，一手抓激素煎熬，在大约五千万年时间里，都是脊椎动物。事实上的最高指挥中心。然而这个体系终究是没能一以贯之。在距今四点四亿多年前的奥淘寂寞大灭绝事件，让地球海洋迎来了一轮大洗牌。借助于大灭绝之后的全力真空，一部分假咒语开始向着熟练的更上游进发。由于不待昼伏夜出，他们的感官迎来了大爆发。但是你前段他们的头部出现了专门负责容纳嗅觉，感受器的鼻孔嗅觉也随之突飞猛进，这反过来又刺激了端脑的进一步膨胀。同时，由于在白天活动眼睛的重要性愈发提升。而丘脑作为一个原本用来处理嗅觉的脑区，并不善于处理视觉。这种实时多线程任务，于是不得不还正宇原本负责处理视觉的钟某抓着这个历史机遇，终脑里面负责视觉的神经细胞急剧扩编，很快膨大，出一个专门负责高效处理视觉信息的脑区被称为视顶盖。这个时候马丘脑的地位就比较尴尬了，视觉信息它处理不了。而嗅觉信息的处理权被端脑夺走后，就从没拿回来过，结果就是丘脑的权利被渐渐架空，而且在端脑和视顶盖一前一后的夹逼下，其相对体积也不断缩小，渐渐沦为一个纯粹负责感觉信息上传下达的秘术。于是，脊椎动物的脑进入一个短暂的二d共治时期，惊慌狂喜了。当此之时，视觉无疑是最重要的感觉，让视顶盖也成了最炙手可热的脑区。而嗅觉虽然地位下降，但负责嗅觉端脑从一开始就主导了脊椎动物神经中枢的底层架构，而其悠久的历史也使其中神经细胞的数量与种类皆更胜一筹。如果按照这个剧本走下去，大同究竟会归于视觉党，还是嗅觉党或是像节肢动物那样走向共和也有未可知。然而就是在这个节骨眼上突然横生变数。首先再跟大家梳理一下脊椎动物的脑部原始形态是三段脑前脑负责嗅觉中脑负责视听觉后脑负责运动。现在前脑衍生出了端脑丘脑下丘脑中脑衍生出了是顶盖，是不是漏了水啊？对，这后脑好像还在一边打酱油呢。对于夺取大权这件事情，后脑从一开始就没戏，毕竟运动属于执行端，感觉才是决策端。后脑虽然也会收到一些全身温痛、出压之类的感觉信息。但这些相比于视觉、嗅觉，以及后来的听觉都过于次要，而头部的关键感觉信息则根本不会经过后脑，然而却偏偏是后农主导了场试秀党真的的历史走向原本咱们脊椎动物因为有脊索加持，运动模一直比较简简单，所以负责运动的后脑也长期无所作为。但随着一只脊椎动物的崛起，一切都变了，那就是骨甲鱼。相比于其他脊椎动物，骨甲鱼有两个明显的特征，一个在外表就是这个藕鳍原本脊椎动物只简单的摆动尾巴。但是随着偶鳍的出现，我们的祖先可以花式游泳的，而另一重变革在内部就是髓窍。简单来说，原本身体外周的感觉，运动神经都是基本裸露的，而髓翘，就是在这些裸露的神经上，按照一定规律包裹上的另一种细胞，而这些细胞会在某种程度上充当导线的作用，将神经纤维表面离子流动，造成电压变化，快速传导到远处打开那里的电压敏感性离子通道，这样神经信号就实现某种跳跃式传导。如果还用之前多米诺骨牌做比喻，就相当于在多米诺骨牌队列中插入了一些推导。这些多米诺骨牌倒下时，直接推动推杆将较远处的多米诺骨牌直接推导。而不用一张一张导过去，说起来神经传导速率就会大大提升。根据测算，在人体中裸露的神经纤维神经信号传输速率大概是一米每秒左右。但那些拥有髓翘的神经纤维传输速率最快可达一百米。每没有，我在几年前说过，随撬是脊椎动物崛起的关键。但最近查资料的时候，我发现这事儿其实没那么简单，其实髓叫演化并不复杂，神经细胞很长，这就意味着神经纤维很难及时获得来自胞体的养分。所以本来就需要一些别的细胞在附近供养它们，而这些细胞只需简单改造，就能变成髓壳。所以髓壳在动物界中其实并不罕见，节肢动物、环节动物都有髓壳，像诸如对虾这些动物甚至演化出了比脊椎动物结构更复杂，传输效率也更快的髓鞘。但是只有脊椎动物能发挥出髓鞘，或者说高速神经的真正潜力，这是我们脊髓的横截面可以看见其中有相当大的面积是白色的。如果我们用显微镜观察这些白色的部分，我们就会发现它其实是巨量的神经纤维汇聚成的树。因为我们的神经架构是先中央后地方，这就有条件，让我们有巨量的神经纤维，可以从脑部出发，直接延伸到身体末端，或者说在你的体内一个神经细胞，很可能它的包体在你的脑子里。但是其神经纤维却一路延伸到你的脚底板。这个神经细胞如果拉直的话，可以比你本人还长。而这样的神经细胞在脑全不下乡的节肢动物、软体动物直流当中极为罕见。他们体内绝大多数外周神经都只为本体界的是宁杰负责。因此，髓翘可以让我们脊椎动物的脑得以极为高效，上传下达感运动信息。而对别的动物来说，绝大多数信息都只能一级一级神经节的上传下达基层，再怎么高效也带不动。这种小偷公司似的是林中枢，而呕脐和髓翘，让脊椎动物的运动变得极剧复杂，也反过来推动运动控制中心，耗脑，走上了改革的快车道。在短短一千多万年间后脑急剧膨胀，它直接连接身体外，周神经的部分膨大成了一个身体。感觉运动信息分流中心被称为脑桥，而在脑桥背侧，更是几乎从无到有的构建起了一个包含巨量神经元与惊人复杂度的专门负责精细调控，运动脑区被称为小脑。只有今天我们全身几乎所有肌肉上都会有直点小脑的神经纤维。而且虽然看上去小脑比我们大脑体积小，很是，但小脑当中包含的神经细胞数量却和大脑基本一致。由于无核鱼类头部还不够发达，高科猛进的强壮成挤尽手段，挤进一切可得的空间中。而这被后来的某件事儿埋下了伏笔。在巨星四眼二亿年前，脊椎动物终于完成了演化史上最重要的改革，有核鱼类出现了夏河的出现。对于我们绝不仅仅是有了和平的武器那么简单，更关键的是，下颌让我们在原来咽腔之前又多了一节名叫口腔的空间。那么相应的头部连在其中的脑子终于又能向前开图了。那么位于最前端的端脑，自然又是近水楼台手，最快毫不客气的迅速膨大。而反观他的竞争，对手是顶盖向前，被端倒挡着向后，还被小脑挤着，只能靠头部的整体增大，往背侧争取少量发展空间。而且屋漏偏逢龙卷风摧毁停车场。随着脊椎动物在各种生态会秒、天秒地秒、空气全新的生活方式以及与其他动物的激烈竞争，促使我们的祖先快速发展出了更强悍的感官与更大的体型。原本的视觉，只要知道物体的形状大小，粗略判断一下是敌害食物还是中立就可以了。但现在我们面对挑战更多样了，视觉辨认得明确，距离方位轮廓、颜色亮度最好还能精确辨别出看到的具体是什么东西。但是顶该分享空间真的是一滴也没有了。趁你病要你命端脑，抓着这个机会，就利用他长期积累的神经细胞多样性，开始搞事情。一部分新型神经细胞开始在头部的新空间中构建起了更复
